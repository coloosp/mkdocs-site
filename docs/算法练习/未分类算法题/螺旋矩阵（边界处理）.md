# 螺旋矩阵
[螺旋矩阵](https://www.lanqiao.cn/problems/156/learning/?page=1&first_category_id=1&name=矩阵)
很棒的边界处理，应该学习
```
#include <iostream>
#include <vector>
using namespace std;
int main()
{
int n,m,r,c;
cin >>n>>m;
cin >>r>>c;
vector <vector<int>> matrix (n,vector<int>(m,0));
int u=0;
int d=n-1;
int left=0;
int right=m-1;
int count=1;
while(true){
  for(int i=left;i<=right;i++){
    matrix[u][i]=count++;
  }
  if(++u >d) break;

  for(int i=u;i<=d;i++){
    matrix[i][right]=count++;
  }
  if(--right <left) break;

  for(int i=right;i>=left;i--){
    matrix[d][i]=count++;
  }
  if(--d <u) break;

  for(int i=d;i>=u;i--){
    matrix[i][left]=count++;
  }
  if(++left >right) break;

}

  cout <<matrix[r-1][c-1]<<endl; 


 return 0;

}

```
## 解析
我们在规定一组动态移动的边界，当我们处理完某一行（列）我们移动相应边界，并且判断边界是否重叠
