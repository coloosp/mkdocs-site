首先明确目的，删除元素一定不是一个优秀的策略，实际上我们的目标是用n-最长的接龙数组，剩下的值都应该删去

确定dp数组的含义：DP~p~指的是对于ai元素为止，以p为结尾的最长的接龙数组
明确递推公式：
设ai的头元素为p，尾元素为q
如果我们需要选取当前元素ai，那肯定用ai的头元素接上前面的尾元素
即dp~p~+1
假如我们不选取ai，也就是不接上前面的元素，我们仍需要维护直到ai为止的dp~q~

有dp~p~=max(dp~p~,dp~q~+1)
```
#include <iostream>
# include <vector>
# include <queue>
#include <algorithm>
# include <unordered_map>
# include <unordered_set>
// #include <bits/stdc++.h>
using namespace std;


int main() {
	int n;
	cin >> n;
	string s;
	vector <int> dp(10,0);
	for (int i = 0; i < n;i++) {
		cin >> s;
		int len = s.size();
		dp[s[len - 1] - '0'] = max(dp[s[len - 1] - '0'], dp[s[0]-'0'] + 1);
	}

	int maxn = 0;
	for (int i = 0; i <=9; i++) {
		maxn = max(dp[i], maxn);
	}

	cout << n - maxn << endl;



	return 0;
}
```
