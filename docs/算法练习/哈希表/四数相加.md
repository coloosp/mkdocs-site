# 四数相加
[四数相加](https://leetcode.cn/problems/4sum-ii/description/)
看起来像是两数之和的加强版，实际上理解思路不算复杂，我们可能都在中学物理课上学过“整体法”，那么把数组1和数组2划分成一个整体，数组3和数组4划分为一个整体如何呢？
## 代码
```
class Solution {
public:
    int fourSumCount(vector<int>& nums1, vector<int>& nums2, vector<int>& nums3, vector<int>& nums4) {
        unordered_map <int,int> map;
        int cout =0;
        for(int a:nums1){
            for(int b:nums2){
                map[a+b]++;
            }
        }
        for (int c:nums3){
            for(int d:nums4){
                int target =0-(c+d);
                if(map.find(target) != map.end()){
                    cout+=map[target];
                }
            }
        }
        return cout;
    }
};
```
## 解析
和两数之和差不多，我们拿着已经存储好数组1和数组2的map数组在数组3和数组4里面找，如果能找到匹配则计数增加相应次数。需要注意的是，我们实际上是记录的数组1和数组2的和，寻找的是数组3和数组4的和。
