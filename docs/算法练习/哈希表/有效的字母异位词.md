# 有效的字母异位词
[有效的字母异位词](https://leetcode.cn/problems/valid-anagram/description/)
哈希表，来判断两个字符串是否组成的字母相同，这个意思。
哈希表的本质是映射与存储，存储什么呢？一般来说可能是次数，标记，指针等等，这里我们的哈希表是存储的某个字母出现的频次。
## 代码
```
class Solution {
public:
    bool isAnagram(string s, string t) {
        int hash[26];
        for(char c:s){
            hash[c-'a']++;
        }
        for(char c:t){
            hash[c-'a']--;
        }
        for(int i=0;i<26;i++){
            if(hash[i] != 0)
                return false;
        }
        return true;



    }
};
```

## 解析
可能第一次让我做，会制作一张map或者set，但这样过于麻烦并且检查效率不高，实际上对于这种记录字母的，我们都可以使用数组很好地记录，可能后面会遇到前缀和匹配，也是利用相同的思路。这里是规定将a放在数组“0”位置上，然后25位置是z，利用数组的特性实际上实现了map的功能。
