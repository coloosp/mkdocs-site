# 快乐数
[快乐数](https://leetcode.cn/problems/happy-number/)
乍一看像是C语言入门实验课做的玩意，实际上还是哈希表，当我们检查到和在set中出现时，这个结果就会无限循环。
## 代码
```
class Solution {
public:
    int getsum(int n)
        {
            int sum=0;
            while(n){
                sum+=(n%10)*(n%10);
                n=n/10;
            }
            return sum;
        }

    bool isHappy(int n) {
        unordered_set <int> set;
        while (1){
            int sum=getsum(n);
            if(sum==1){
                return true;
            }
            if(set.find(sum) !=set.end() ){
                return false;
            }
            set.insert(sum);
            n=sum;
        }
    }
};
```
## 解析
快乐就是能够在有限的次数下解决一个问题，我们需要的快乐数是不允许循环的，如果出现循环就以为着会无限循环下去，set就是为了标记每次出现的数字，题目挺简单的，获取一个数字的每一位数可能需要学一下（如果以前不会的话）
