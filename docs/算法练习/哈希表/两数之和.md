# 两数之和
[两数之和](https://leetcode.cn/problems/two-sum/submissions/606608325/)
梦开始的地方，暴力搜索就无需再提了，使用哈希表来完成吧！这里我们使用的是map结构，map结构可以实现任意两个数据类型的映射，这里我们完成的是"值"与"键"的映射
## 代码
```
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map <int,int> map;
        
        for(int i=0;i<nums.size();i++){
        if(map.find(target-nums[i]) != map.end()){
            return {map[target-nums[i]],i};
        }    
        map.insert({nums[i],i});
        }
        return {};
    }
};
```
## 解析
非常高效且高明的搜索方式，使用哈希表一趟搜索就能完成，我们使用了值与键（下标）的映射，当我们没有在map中找到对应值的时候，就把这对值键对插入map中，找到的话，就根据搜索值来返回对应键和当前i值。
