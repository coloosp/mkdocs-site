# 赎金信
[赎金信](https://leetcode.cn/problems/ransom-note/)
和有效字母异位词很像，嗯，相信各位也能很轻松地解决这道哈希表的题目吧
## 代码
```
class Solution {
public:
    bool canConstruct(string ransomNote, string magazine) {
        int hash[26];
        for(char c:magazine){
            hash[c-'a']++;
        }
        for(char c:ransomNote){
            hash[c-'a']--;
            if(hash[c-'a']<0)
                return false;
        }
        return true;





    }
};
```
## 解析
像这种需要记录字母的题目，使用数组类型的哈希表即可啦，没什么好讲的
