# 翻转字符串的单词
[翻转字符串的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)
素质最差的一集，我们来理解题意，题目的要求有两个，一、清除多余空格，二、翻转每个单词。所以说字符串会考验边界处理能力很高，我和大家还得继续练
## 代码
```
class Solution {
public:
    void reverse(string &s,int a,int b){
        for(int i=a,j=b;i<j;i++,j--){
            swap(s[i],s[j]);
        }
    }

    string reverseWords(string s) {
        int index=0;
        for(int i=0;i<s.size();i++){//擦除多余空格
            if(s[i]!=' '){
                if(index >0 ){//需要我们手动填入空格
                    s[index++]=' ';
                }
                while(i <s.size() && s[i]!=' '){
                    s[index++]=s[i++];
                }
            }
        }
        s.resize(index);
        int slow=0;
        reverse(s,0,s.size()-1);
        for(int fast =0;fast <s.size() && slow <s.size();fast++){//翻转字符串
            while(fast <s.size() && s[fast]!=' '){
                fast++;
            }
            reverse(s,slow,fast-1);
            slow=fast+1;
            fast++;
        }
        return s;


    }
};
```
## 解析
清除多余空格和翻转字符串我们实际上都用到了双指针（可见其重要性），对于擦除空格，实际上也可以分为两部分，当我们检测到字符时一次性填入，但是当我们再次检测到字符时，说明是经历了两个单词，手动填入一个空格。
对于单词的翻转，相信细心的各位发现了，实际上是我们整体翻转，再把每个单词翻转。
