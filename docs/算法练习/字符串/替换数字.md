# 替换数字
[替换数字](https://kamacoder.com/problempage.php?pid=1064)  
当我们遇到这种需要删除的数组或者字符串，开辟一个新的数组或许是一个好方法。我们先试试开辟数组的方式
## 代码
### 开辟新数组
```
# include<iostream>
# include<string>
using namespace std;
 int main(){
    string s;
    cin >>s;
    string t;
    for(int i=0;i< s.size();i++){
        if(s[i] >='0'&&s[i]<= '9'){
            t+="number";
        }
        else{
            t+=s[i];
        }
    }
    cout <<t<<endl;

 }
```
那让我们试着在原数组上操作看看？
### 双指针
```
# include<iostream>
# include<string>
using namespace std;
 int main(){
    string s;
    cin >>s;
    int left=s.size()-1;//左指针是原来的数组边界
    int count =0;
    for(int i=0; i<s.size();i++){//检查原数组的数字个数
        if(s[i]>='0' && s[i] <='9')
        count ++;
    }

    s.resize(s.size()+5*count);//扩容s，因为是删去数字在增加“number”
    int right=s.size()-1;//扩容后的数组边界
    while(left >=0){
        if(s[left]>='0' && s[left] <='9'){//检测到是数字，反向添加"number"
            s[right--]='r';
            s[right--]='e';
            s[right--]='b';
            s[right--]='m';
            s[right--]='u';
            s[right--]='n';
        }
        else{
            s[right--]=s[left];
        }

        left--;
    }
    cout << s<<endl;


 }


 }
```
## 解析
对于开辟新数组来说，只需要我们检查到数字，就向新数组里面添加"number"，反之则直接添加原位置元素。
而对于双指针法来说，我们需要先检测原

数组，因为我们是在原字符串里做修改，首先是确定字符串的新长度，然后我们需要从后往前检测，当是字母就直接添加，检测到数字则需要反向添加"number"。
为什么不是从前往后呢？大家可以试想，从前往后，我们每次检测到数字，就需要把后面的字母移动，这样会极大增加时间复杂度
