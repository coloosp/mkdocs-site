当然，这道题如果没有限制空间就相当简单，限制了空间实际上就只能对s做出操作，但是这道题并非"交换",所以双指针可能会出现覆盖的情况
## 代码（渴望胜利版）
```
# include <iostream>
# include <string>
using namespace std;
int main(){
    int k;
    cin >>k;
    string s;
    cin >>s;
    int index=s.size()-k;
    for(int i=0;i<s.size();i++){
        cout <<s[index%(s.size())];
        index++;
    }
    cout <<endl;

}
```
## 解析
但是，谁告诉你一定要输出字符串了？答案是我们只需要按照特定顺序输出即可！

## 代码
```
# include <iostream>
# include <string>
using namespace std;
void reverse(string&s,int a,int b){
    for(int i=a,j=b;i<j;i++,j--){
        swap(s[i],s[j]);
    }
}
int main(){
    int k;
    cin >>k;
    string s;
    cin >>s;
    reverse(s,0,s.size()-1);
    reverse(s,0,k-1);
    reverse(s,k,s.size()-1);
    cout <<s<<endl;
}
```
## 解析
还记得我们在翻转单词那里学到的吗，实际上我们可以把s字符串看成两部分，先整体翻转，再对每一部分局部翻转
