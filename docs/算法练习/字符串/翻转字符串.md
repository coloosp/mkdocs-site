# 翻转字符串
[翻转字符串](https://leetcode.cn/problems/reverse-string/description/)
奖励关罢了，相信大家都可以独立完成的
## 代码
```
class Solution {
public:
    void reverseString(vector<char>& s) {
        for(int i=0,j=s.size()-1;i<j;i++,j--){
            char tmp =s[i];
            s[i]=s[j];
            s[j]=tmp;
        }




    }
};
```
## 解析
前后指针交换就完了，先麻烦可以使用swap函数
