# 电话号码的字母组合
[电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)
和前面思路类似，但这里需要明确每一次遍历是在哪一个字符串数组里面，实际上前面我们都在同一个数组遍历，在这里得思考更多

## 代码
```
class Solution {
public:
    vector <string> result;
    string s;
    const string letterMap[10]={"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};
    void backtracking (string &digits,int index){
        if(s.size() == digits.size()){//这里写index也可以
            result.push_back(s);
            return ;
        }

        int number=digits[index]-'0';//强制转化，获取对应索引的对应数字
        string letters =letterMap [number];//获取当前数字代表的字符串数组
        for(int i=0;i< letters.size();i++){
            s.push_back(letters[i]);
            backtracking(digits,index+1);
            s.pop_back();
        }





    }




    vector<string> letterCombinations(string digits) {
        if(digits.size()==0){
            return result;
        }
        backtracking(digits,0);
        return result;
    }
};
```

## 解析
实际上题目要求对于每一个数字对应的字符串进行组合，和前面的[1,n]不同，这里每一次遍历或许都在不同的字符串数组之中，不过万变不离其宗嘛，本质还是对于各组合的穷举遍历
