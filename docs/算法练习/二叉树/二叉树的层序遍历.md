# 二叉树的层序遍历
[二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/)
二叉树的层序遍历是我认为二叉树遍历里最好用最容易理解的，获得的结果还可以很容易找到二叉树的深度，叶子节点数，判断二叉树种类等特点
我们在二叉搜索树中使用了队列，它可以有效区分该层和下一层的节点
## 代码
```
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        queue <TreeNode*> que;
        vector <vector <int>> result;
        if(root==nullptr) return result;
        que.push(root);
        while(!que.empty()){
            vector <int> vec;//记录当前层的长度
            int size=que.size();
            while(size--){
                TreeNode *cur=que.front();
                vec.push_back(cur->val);
                que.pop();
                if(cur->left) que.push(cur->left);
                if(cur->right) que.push(cur->right);//加入该节点的左右节点到队列尾
            }
            result.push_back(vec);
        }
        return result;
    }
};
```
## 解析
二叉树的层序遍历还是相当容易理解的，我们根据size来获取当前层的节点数，处理后包装成vec加入结果中，同时每一次处理都可以将该节点的左右子节点加入到队列中，为下一层处理做准备
