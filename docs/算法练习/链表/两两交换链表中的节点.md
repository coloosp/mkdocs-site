# 两两交换链表中的节点
[交换链表节点](https://leetcode.cn/problems/swap-nodes-in-pairs/description/)
模拟题，很考验对于链表的掌握程度，一定要画图做，不然很容易乱，嗯，多试几次吧
## 代码
```
class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        ListNode* DummyHead=new ListNode(0);
        DummyHead->next=head;
        ListNode *cur=DummyHead;
        while(cur->next !=nullptr && cur->next->next!=nullptr){
            ListNode *tmp=cur->next;
            ListNode *tmp1=cur->next->next->next;
           cur->next=tmp->next;
           tmp->next->next=tmp;
           tmp->next=tmp1;

            cur=cur->next->next;
        }
        return DummyHead->next;



    }
};
```
## 解析
这里还是建议使用虚拟节点来处理，好处有两个，一是便于寻找到头结点以便于返回，二是便于管理节点。我们不仅要把链表交换成功，还要正确的返回头结点，这是隐性的条件，一定要记得按照顺序来处理每一部分
![alt text](image.png)
1. 将cur指向2节点
2. 将2节点指向1节点
3. 将1节点指向3节点
可以看到整个过程需要包含4个节点，前置节点cur，后置节点tmp1，需要交换的两个节点tmp与tmp->next
