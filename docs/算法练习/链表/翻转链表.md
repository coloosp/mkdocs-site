# 翻转链表
[翻转链表](https://leetcode.cn/problems/reverse-linked-list/)
对于一个单向链表来说，翻转链表就是把节点的next都指向前面一个节点，明确这条思路就比较好做了
双指针法
## 代码
```
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* pre=nullptr;
        ListNode* cur=head;
        ListNode* tmp;
        while(cur){
            tmp=cur->next;//注意临时保存cur后面的节点，否则断开指针就搜寻不到了
            cur->next=pre;
            pre=cur;
            cur=tmp;
        }
        return pre;
    }
};
```
## 解析
我们需要明确一个概念：确定两个前后两个节点就可以将后一个节点cur指向前面一个节点pre。不断地移动cur和pre直到pre到达最后一个节点，cur会到达之前最后一个节点指向的nullptr。
