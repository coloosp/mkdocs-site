# 用队列实现栈
[用队列实现栈](https://leetcode.cn/problems/implement-stack-using-queues/description/)
居然还给我两个队列？用一个足够了！
队列比起栈的优点是队列可以把弹出的元素储存在另一端，而栈因为只有一个出入口不能实现
## 代码
```
class MyStack {
public:
    queue <int> que;
    MyStack() {
        
    }
    
    void push(int x) {
        que.push(x);
    }
    
    int pop() {
        int size=que.size();
        size--;
        //先弹出再放入即可
        while(size--){
            int temp=que.front();
            que.pop();
            que.push(temp);
        }
        int result =que.front();
        que.pop();
        return result;
    }
    
    int top() {
        return que.back();
    }
    
    bool empty() {
        return que.empty();
    }
};

/**
 * Your MyStack object will be instantiated and called as such:
 * MyStack* obj = new MyStack();
 * obj->push(x);
 * int param_2 = obj->pop();
 * int param_3 = obj->top();
 * bool param_4 = obj->empty();
 */
```
