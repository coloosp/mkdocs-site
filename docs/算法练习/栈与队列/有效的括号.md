# 有效的括号
[有效的括号](https://programmercarl.com/0020.有效的括号.html#算法公开课)
记得以前做了一样的题目，当时不知道栈这种数据结构，检测过的字符串不知道存储在哪里，这道题虽然是简单题，但是设计的相当精妙
## 代码
```
class Solution {
public:
    bool isValid(string s) {
        if(s.size() %2 !=0){//剪枝
            return false;
        }
        stack <char> st;
        for(int i=0; i<s.size();i++){
            if(s[i]=='(') st.push(')');
            else if(s[i]=='{') st.push('}');
            else if(s[i]=='[') st.push(']');
//当遇到左括号就向栈中添加右括号
            else if(st.empty() || st.top() !=s[i])//数组没遍历完而栈空了，说明多了右括号，栈顶元素匹配错误则说明括号顺序错误
                return false;
            else if(st.top() == s[i]){//当匹配的一对括号移除
                st.pop();
            }
        }
        return st.empty();
    }
};
```
## 解析
栈是一种数据容器，它的功能就是存储，这里是存储什么呢？我的理解是栈在存储“**当检查到左括号时期望找到右括号的值与它们的顺序**”，当我们检查到'('，实际上下一个右括号我们希望是'(',然后我们检查到'['，我们会希望检查的右括号顺序变成了'])'。
对于一个数据结构，它所存储的数据绝对没有想象的那么简单，它可以是指针，数据类型，新的数据结构，下标等等
