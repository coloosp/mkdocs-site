# 用栈实现队列
[用栈实现队列](https://leetcode.cn/problems/implement-queue-using-stacks/description/)
终于到栈和队列的部分了，栈和队列算是一对难兄难弟了，一个先进后出，一个先进先出，看起来很简单的数据结构，但是不要小看他们
## 代码
```
class MyQueue {
public:
    stack <int> stack_in;
    stack <int> stack_out;

    MyQueue() {
        
    }
    
    void push(int x) {
        stack_in.push(x);
    }
    
    int pop() {
        //出栈为空的时候需要在入栈里面进货
        if(stack_out.empty()){
            //需要将入栈全部倒出来才能找到队列顶端元素
            while(!stack_in.empty()){
                stack_out.push(stack_in.top());
                stack_in.pop();
            }
        }
            int result=stack_out.top();
            stack_out.pop();
            return result;
    }
    
    int peek() {
        //因为是在类中，所以直接调用即可
        int result=this->pop();
        stack_out.push(result);
        return result;
    }
    
    bool empty() {
        //只有入栈和出栈都为空才能判空
        return stack_in.empty() && stack_out.empty();
    }
};

/**
 * Your MyQueue object will be instantiated and called as such:
 * MyQueue* obj = new MyQueue();
 * obj->push(x);
 * int param_2 = obj->pop();
 * int param_3 = obj->peek();
 * bool param_4 = obj->empty();
 */
```
## 解析
记住栈和队列的数据存储特点，想要取出队列头的数据，也就是栈底的数据，就需要将之前的元素一个一个拿出来，所以我们需要分出栈和入栈两个，比较简单的代码，应该很容易看懂
