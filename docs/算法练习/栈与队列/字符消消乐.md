# 字符消消乐
[消消乐](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/description/)
实际上就是消消乐，对于匹配问题，首先应该想到栈
## 代码
```
class Solution {
public:
    string removeDuplicates(string s) {
        stack <int> st;
        for(int i=0;i<s.size();i++){
            if(!st.empty() && st.top()==s[i]){
                st.pop();
                continue;
            }
            st.push(s[i]);
        }
        string result="";
        while(!st.empty()){
            result+=st.top();
            st.pop();
        }
        reverse(result.begin(),result.end());
        return result;



    }
};
```
## 解析
这里栈有两种处理方式，一种是没有匹配成功，压入栈中；一种是匹配成功，弹出栈顶。栈可以暂存需要处理的数据，等到需要处理的时候再一并弹出，以后我们还会遇到的
