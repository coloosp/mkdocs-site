# 逆波兰表达式
[逆波兰表达式](https://leetcode.cn/problems/evaluate-reverse-polish-notation/)
逆波兰表达式是一种后缀形式的表达方式，具体可以百度了解。
先说结论，逆波兰表达式实际上是遇到运算符号就处理前面两位数字
诶？如果说我们是先存储再处理，那岂不是先进行数据的存储如果达成某种条件就对存储的数据进行处理吗（废话）
## 代码
```
class Solution {
public:
    int evalRPN(vector<string>& tokens) {
        stack <long long> st;
        for(int i=0;i<tokens.size();i++){
            //当检测到运算符则处理
            if(tokens[i]=="+" || tokens[i]=="-" || tokens[i]=="*" || tokens[i]=="/"){
                long long num2=st.top();
                st.pop();
                long long num1=st.top();
                st.pop();
                if(tokens[i]=="+") st.push(num1+num2);
                if(tokens[i]=="-") st.push(num1-num2);
                if(tokens[i]=="*") st.push(num1*num2);
                if(tokens[i]=="/") st.push(num1/num2);
            }
            //当检测到数字则存储
            else{
                st.push(stoi(tokens[i]));
            }
        }
        long long result=st.top();
        return result;



    }
};
```
