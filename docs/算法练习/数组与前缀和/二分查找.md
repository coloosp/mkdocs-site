# 二分查找
[二分查找](https://leetcode.cn/problems/binary-search/)
毕竟二分查找还是老东西里面最有用的，后面很多搜寻方式可能都会用到，重点是理解边界值的区分
## 代码
```
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int left=0;
        int right=nums.size()-1;
        while(left <=right){
            int mid=(left+right)/2;
            if(nums[mid]>target)
                right=mid-1;
            if(nums[mid]<target)
                left =mid+1;
            else if(nums[mid]==target)
                return mid;
        }
        return -1;
    }
};
```
## 解析
| 区间定义 | 循环范围(while) | 左区间查找 |  右区间查找  |
| :------: | :-------------: | :--------: | :---------: |
| 左闭右闭 |   left<=right   | left=mid+1 | right=mid-1 |
| 左闭右开 |   left<right    | left=mid+1 |  right=mid  |

左闭右开可以将right理解为在数组外一位，不过我还是推荐使用左闭右闭的理解形式，嗯，也会也强制规定自己使用左闭右闭的形式。
