# 移除元素
[移除元素](https://leetcode.cn/problems/remove-element/submissions/604587634/)
利用快慢指针法来解决本题，这也是我们之后常用的一种解题思路
 ## 代码
 ```
 class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int slow=0;//慢指针
        for(int fast=0;fast<nums.size();fast++){//快指针会一直走
            if(nums[fast]!=val){
                nums[slow]=nums[fast];//当检测到不是目标值时慢指针往前走
                slow++;
            }
        }
    return slow;//慢指针在fast检索到目标值时不会走，所以值一定是原数组长度减去目标值在数组中的个数
    }
};
 ```
 ## 解析
 快慢指针的本质就是快慢（这不是废话吗），利用特定条件导致两者的速度不同，从而可以通过两者区分一些元素或者一些情况，此处就是通过慢指针在要删除的目标元素不走来无视掉目标元素，从而达到删除的效果，这可能也给我们一些启示，“删除”的本质可能不意味着擦除，覆盖，无视，都可能表示删除，此事在链表中亦有记载
