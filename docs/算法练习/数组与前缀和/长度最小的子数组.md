# 长度最小的子数组
滑动窗口，话说虽然概念不一样，最近学的计网也有这个滑动窗口来着
## 代码
```
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int result=INT32_MAX;//最终的长度值，设置为最大值是方便后面比较
        int sum=0;//滑动窗口值
        int i=0;//左窗口指针
        int subL=0;//滑动窗口大小
        for(int j=0;j<nums.size();j++){//右窗口指针
            sum+=nums[j];
            while(sum >=target){//注意这里是while而不是if，我们可能更新不止一次
            subL=j-i+1;
            result=min(result,subL);
            sum-=nums[i++];
            }
        }
        if(result==INT32_MAX)
            return 0;
        else
            return result;
    }
};
```
## 解析
滑动窗口在实现层面上有点类似于快慢指针，但滑动窗口更强调在窗口区间中的属性，在这里我们会统计窗口大小和窗口值综合，注意不是说滑动的最后一个窗口就是最小窗口，是我们会把所有符合窗口的窗口大小值进行统计
