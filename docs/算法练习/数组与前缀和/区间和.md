# 区间和
[区间和](https://kamacoder.com/problempage.php?pid=1070)
标准的前缀和题目，可能画画图更容易理解些，也很好理解，要知道a到b的区间和，实际上知道从数组开头到b的总和减去从数组开头到a的总和即可
## 代码
```
# include<iostream>
# include <vector>
using namespace std;
int main(){
    int n;
    int num;
    vector <int> prefix_sum (n+1,0);
    cin >> n;
    for(int i=1;i<=n;i++){
        cin >>num;
        prefix_sum[i]=num+prefix_sum[i-1];
    }
    
    int a,b;
    while(cin >>a>>b){
        int result=prefix_sum[b+1]-prefix_sum[a];
        cout <<result<<endl;
    }
}
```

## 解析
构造一个前缀和数组，因为我的前缀和是往后挪了一位（因为a=0的时候不能再减一位了），用后面一块区间和减去前面一块区间和即可
