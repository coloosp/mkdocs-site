## Cmake安装
CMake是一个跨平台的编译(Build)工具,可以用简单的语句来描述所有平台的编译过程。

CMake能够输出各种各样的makefile或者project文件，能测试编译器所支持的C++特性,类似UNIX下的automake。

安装完Ubuntu后，可以在控制台中输入一下命令直接安装  
`sudo snap install cmake --classic`
可以通过以下命令检查是否安装成功  
`cmake --version`

## 正式编译与运行
按照以下方式运行即可  
### 创建一个build文件夹并进入
`mkdir build && cd build`

### 生成构建文件
`cmake ..`

### 构建项目
`make`

### 如果要清理生成的可执行文件
` make clean`

### 运行
`./main`
