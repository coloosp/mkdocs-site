# 课程表
[课程表](https://leetcode.cn/problems/course-schedule/?envType=study-plan-v2&envId=top-100-liked)

本题需要使用拓扑排序的思想解决，或者说实际上是在判断图是否有环，这也是经典算法了

判断环其实有两种实现方式(还有一种并查集比较麻烦)，广搜拓扑排序，深搜
各位在数据结构书本上一个学习过广搜的思路
作者会把两种方法都实现的

## 广搜拓扑排序
### 解析
实现思路如下
1. 构造邻接表（这里邻接表显然是更好的选择),记录所有节点的入度(也就是需要学习的前置课程数)
2. 构造辅助队列，将所有入度为0的课程加入队列中
3. 弹出队首，并将其所有后置课程(也就是需要学习队首课程才能学习的课程)入度-1
4. 如果弹出队首数等于课程数说明全部都可以学完

```
class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        //使用了拓扑排序的思维
        int count=0;
        vector <int> indegree(numCourses,0);//入度数组
        vector<vector<int>> grid(numCourses); //实际上是邻接表

        //在图里面应该是pre[1]->pre[0]
        for(auto pre : prerequisites)
        {
            indegree[pre[0]]++;//入度+1
            grid[pre[1]].push_back(pre[0]);//构造图
        }

        queue <int> Q;//辅助队列，存储当前不需要前置课程就可以学习的课

        for(int i=0;i<numCourses;i++)
        {
            if(indegree[i]==0) //如果入度为零就入队
            {
                Q.push(i);
            }
        }

        while(!Q.empty())
        {
            int v=Q.front();
            Q.pop();
            count++; //学习成功！
            //弹出入度为零的点并把其相连的线移除
            //也就是对其相邻的点入度-1
            for(int i=0;i<grid[v].size();i++)
            {
                indegree[grid[v][i]]--;
                if(indegree[grid[v][i]]==0)
                    Q.push(grid[v][i]);
            }
        }

    if(count!=numCourses) //没学习完所有课
        return false;

        return true;

    }
};
```

时间复杂度O(V+E) 节点数+边数
空间复杂度O(V+E) 节点数+边数

## 深搜染色法
### 解析
貌似力扣的题解都是带bool返回值的，不过我的写法应该更好理解一些(请看起来选择，大佬的题解显然更加简练更需要对回溯的掌握)

我们知道深搜就是一条路走到死，我们约定由三种颜色
0 未搜索
1 正在搜索
2 搜索完成

如果各位了解函数栈或者回溯就知道我们的函数在递归时是向下探索结束再向上返回，当还在向下时的节点为正在搜索，而搜索完成时会向上返回，这时的节点为搜索完成节点

### 代码

```
class Solution {
public:
    bool flag=true; //全局变量用来标记环
    void DFS(vector<vector<int>> &grid,vector<int>& used,int v)
    {
        used[v]=1;
        for(int i=0;i<grid[v].size();i++)
        {
            int w=grid[v][i];
            if(used[w]==1) //这说明在搜索过程中遇到了已经搜索过的点
            {
                flag=false;
                return;
            }
            else if(used[w]==0)
                DFS(grid,used,w);

            //如果是used[w]==2,说明对其已经搜索结束过
        }
        used[v]=2; //到这里说明是回溯返回了，标记已经搜索过了
    }

    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        vector<int> used(numCourses,0);
        vector <vector<int>> grid(numCourses);
        for(auto pre:prerequisites)
        {
            grid[pre[1]].push_back(pre[0]);
        }
       
       for(int i=0;i<numCourses;i++)
       {
        //这里有趣的是我们并非从入度为零开始，在任意节点开始都是成立的
        if(used[i]==0) //对所有未搜索节点进行搜索
            DFS(grid,used,i);
       }

       return flag;
    }
};
```


实际上这种方法也可以用来获取拓扑序列，只需要使用栈在`used[w]=2`后写上`stack.push(w)`
最终获取的栈为拓扑排序的倒序

时间复杂度O(V+E)
空间复杂度O(V+E)
