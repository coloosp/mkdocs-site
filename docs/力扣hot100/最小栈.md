# 最小栈
[最小栈](https://leetcode.cn/problems/min-stack/?envType=study-plan-v2&envId=top-100-liked)

应该说是知道思路很容易完成的题目，搞笑的是这道题是作者刷短视频偶然看到的，只看到双栈各位就知道该怎么实现了

## 解析
思路就是维护两个栈
一个栈正常存储(称为普通栈)，一个栈维护最小值(称为最小值栈)

在加入时处理以下情况
普通栈正常加入
最小值栈为空时没的选，加入值即可
不为空对比加入值和最小值栈的栈顶值，加入其中最小的一个

在弹出时正常弹出即可

在获取栈顶时返回普通栈的栈顶即可

在获取最小值时，返回最小值栈维护的栈顶

## 代码
```
class MinStack {
public:
//最小栈需要使用双栈结构，一个栈存储当前最小值，一个栈存储正常值
    stack<int> minst;//最小值栈
    stack<int>st;//普通栈
    MinStack() {
        //并不需要做什么
    }
    
    void push(int val) {
        //普通栈正常加
        st.push(val);
        //最小值栈根据情况选择加入方式

        if(minst.empty())
            minst.push(val);
        else
        {
            int topval=minst.top();
            if(topval < val)
                minst.push(topval);
            else
                minst.push(val);
        }
    }

    
    void pop() {
        //正常弹出即可
        minst.pop();
        st.pop();
    }
    
    int top() {
        return st.top();
    }
    
    int getMin() {
        return minst.top();
    }
};
```
