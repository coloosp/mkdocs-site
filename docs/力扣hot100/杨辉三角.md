## 解析
总之没什么好说的，当前值等于上一层的左上值+右上值
1. **确定dp数组与下标的含义**：triangle[i][j]为i层j位置的值
2. **确定递推公式**：triangle[i][j]=triangle[i-1][j-1]+triangle[i-1][j] 当前值等于左上值加右上值
3. **dp数组初始化** ：初始化首部的triangle[i][0]=1以及尾部的triangle[i][i]=1
4. **确定遍历顺序** :从第一层遍历到最后一层
5. **举例推导dp数组** ：力扣例子已经很明显了

## 代码
```
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>> triangle(numRows);

        for(int i=0;i<numRows;i++)
        {
            triangle[i].resize(i+1);
            triangle[i][0]=1; //首部初始化
            for(int j=1;j<i;j++)
                triangle[i][j]=triangle[i-1][j-1]+triangle[i-1][j]; //状态更新
            triangle[i][i]=1; //尾部初始化
        }

        return triangle;


    }
};
```
