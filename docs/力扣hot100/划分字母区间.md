# 划分字母区间
[划分字母区间](https://leetcode.cn/problems/partition-labels/description/?envType=study-plan-v2&envId=top-100-liked)

**"当断则断"**

##  解析
这道题掌握一个核心思想就可以实现"字母的最远出现位置"

也就是说我们切割字符串有一个逻辑就是必须等到当前切割区间中字母的最远出现位置被包含进去才能进行切割

换句话说，只要我们能够到达当前切割字符串中任意字母的最远出现位置我们就可以进行切割对吧。

接下来完成这道题就很简单了，贪心题找对思想实现都不难
## 代码
```
class Solution {
public:
    vector<int> partitionLabels(string s) {
        //可恶的字符串！
        //最远出现位置？我脑子里面蹦出了这个词语
        vector<int> result;
        vector <int> letterhash(26,0);
        int reach=0;//需要到达的最近位置
        int precult=0;//上一次切割的点
        int curcult=0;
        for(int i=0;i<s.size();i++)
        {
            char c=s[i];
            letterhash[c-'a']=max(letterhash[c-'a'],i);
        }

        for(int i=0;i<s.size();i++)
        {
            char c=s[i];
            reach=max(reach,letterhash[c-'a']);

            if(i==reach)    //当断则断
            {
                curcult=i;
                result.push_back(curcult-precult+1);
                precult=curcult+1;//更新下一个片段起点
            }

        }

        return result;
        
        


    }
};
```

时间复杂度O(n)
空间复杂度O(n)
