# 旋转图像
[旋转图像](https://leetcode.cn/problems/rotate-image/?envType=study-plan-v2&envId=top-100-liked)
答案是注意力惊人

好吧，题目要求我们不能使用其他矩阵来存储，我们就必须原地处理，其实很容易想到需要交换

对于正方形的交换就会想到几种，比如左右翻转，上下翻转，转置这些，稍微尝试一下应该也能想出解决方法，毕竟是思维题，各位可以自行尝试一下

```
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        //注意力惊人，注意到将矩阵由对角线翻转，然后再将左右两边交换就可以实现旋转图像

        int n=matrix.size();
        for(int i=0;i<n;i++){ //对角翻转
            for(int j=i+1;j<n;j++){
                swap(matrix[i][j],matrix[j][i]);
            }
        }

        for(int j=0;j<n/2;j++){ //左右翻转
            for(int i=0;i<n;i++){
                swap(matrix[i][j],matrix[i][n-j-1]);
            }
        }


    }
};
```

时间复杂度 O(n)
空间复杂度  O(1)
