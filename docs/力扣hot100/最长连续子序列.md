# 最长连续子序列
[最长连续子序列](https://leetcode.cn/problems/longest-consecutive-sequence/?envType=study-plan-v2&envId=top-100-liked)

```
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        //1.遍历找出数组最小值与最大值
        //2.构造哈希表，长度为最大值减最小值
        //3.遍历数组将对应数字减去最小值填入数组
        //4.构造dp数组即可，因为只需要返回长度

        if(nums.size()==0)
            return 0;
        int  minnum=INT_MAX;
        int maxnum=INT_MIN;
        for(int i=0;i<nums.size();i++){
            minnum=min(minnum,nums[i]);
            maxnum=max(maxnum,nums[i]);
        }

        vector <int> hash(maxnum-minnum+1,0);

        for(int i=0;i<nums.size();i++)
            hash[nums[i]-minnum]++;

       int result=1;

       vector <int> dp(maxnum-minnum+1,0);
        dp[0]=1;
       for(int i=1;i<hash.size();i++){
        if(hash[i]==0)
            continue;

        if(hash[i-1]>0){
            dp[i]=dp[i-1]+1;
            result=max(result,dp[i]);
        }
        else
            dp[i]=1;
       }

        return result;

    }
};
```

写了这样一段代码，使用了哈希表+动态规划的设计(dp其实只需要两个数就行了)但是内存爆炸过不了，好吧，看起来我们只能使用stl了
也就是集合，对于unordered_set的一个数字的查询只需要O(1)

## 正解
```
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set st(nums.begin(),nums.end());
        int result=0;
        for(int x: st) //对集合遍历,比迭代器遍历要方便一点
        {
            if(st.count(x-1)!=0) //因为我们需要找到每个子序列的开始
                continue;
            int y=x+1; //查找下一个数字是否存在
            while(st.count(y)!=0){
                y++;
            }

            result=max(result,y-x); //更新结果
        
        }

        return result;






    }
};
```  
  
## unordered_set
由于作者也没怎么学习过STL的知识，所以列出来给自己看吧
头文件 ：`#include <unordered_set>`
unordered_set 是 C++ STL 中基于哈希表实现的无序集合容器，核心特性是元素唯一（自动去重）、查询 / 插入 / 删除平均时间复杂度 O (1)

|      操作方法       |                            功能说明                            | 时间复杂度 |             算法题使用场景              |
| ------------------ | -------------------------------------------------------------- | ---------- | -------------------------------------- |
| `insert(val)`      | 插入值`val`，重复值自动忽略，无返回值（或返回pair<迭代器,bool>） | 平均O(1)   | 初始化集合、动态添加元素                |
| `count(val)`       | 查询`val`是否存在，存在返回1，不存在返回0                        | 平均O(1)   | 核心：存在性判断（如`num-1`是否在集合） |
| `find(val)`        | 查询`val`，存在返回指向该元素的迭代器，不存在返回`end()`         | 平均O(1)   | 需获取元素迭代器时使用（比count灵活）   |
| `empty()`/`size()` | `empty()`判断是否为空，`size()`返回元素个数                     | O(1)       | 边界判断（如数组为空时直接返回）        |
