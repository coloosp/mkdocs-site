## 解析
这道题乍一看几乎是无法解决的，让我们模拟出删除，插入和替换绝非易事

首先应该从哪里讲起呢？
我们会以题目的"horse"->"ros"说明
比如"ho"->"ro"怎么转变？答案当然是替换 将h->r即可
那"ror"->"ro"怎么转变，答案是删除后面的r

我们可以做出这样的dp数组，就像我说的，对于多维动态规划，我们应该先思考dp的定义
dp[i][j]假设表示对于word2[0,j)->word1[0,i)需要的最少方法数,这是因为我们会预留第0行与第0列

思考以下情况，加入word2[i-1]==word1[j-1]相同的话，dp[i][j]应该是多少
比如"ho"->"ro" 看起来我们不需要在动两个相同的o，只需要继承'h'->'r'的操作数即可

所以当两字母相同时，我们只需要考虑除末尾字母为前面两子串的操作时(是不是很眼熟)
得出的结论应该是dp[i][j]=dp[i-1][j-1]当前的操作数只取决于前面两子串的操作数，**而前面两子串的操作数之前已经算过了！**

**再来考虑一下两字母不同该怎么做**
两字母不同时我们可以采取三种策略，删除，替换，插入，注意我们只读word1进行以上操作，Word2保持原样就行

先说替换，替换说明当前操作数+1，然后使末尾元素变成一致，当末尾元素一致时，问题就回到了前面子串的操作数是多少
所以更新策略应该是dp[i-1][j-1]+1

在来看看删除word1的当前元素，很好理解，删除了就相当于由j转变为了j-1位置再比较对吧
而dp[i][j-1]前面已经算过了！
所以策略应该是dp[i][j-1]+1

再来看看插入难道插入是dp[i][j+1]+1吗？
这绝对不可能，因为我们不能预知未来发生了什么，我们压根就没算dp[i][j+1]是多少
这里我只能下结论，插入word1一个字母相当于删除word2一个字母，证明其实很麻烦，抱歉我确实没有足够能力证明，各位应该猜到递推了吧
dp[i-1][j]+1

呼，综合起来dp[i][j]=min(dp[i-1][j-1],min(dp[i][j-1],dp[i-1][j]))+1

还没完，我们还需要考虑如何初始化，这也是难点
比如"horse"->""该怎么做？没什么好说的，删除五个元素
反过来""->"ros"，则需要插入3个元素

让我们最后来实现动态规划五步吧！
1. **确定dp数组与下标的含义**：dp[i][j]假设表示对于word2[0,j)->word1[0,i)需要的最少方法数
2. **确定递推公式**：
* 当word2[i-1]==word1[j-1]时，dp[i][j]=dp[i-1][j-1]
* 否则，dp[i][j]=min(dp[i-1][j-1],min(dp[i][j-1],dp[i-1][j-1]))+1
3. **dp数组初始化**:第零行与第零列从零开始递增
4. **确定遍历顺序**：从第一行第一列开始向后遍历 
5. **举例推导dp数组**    

|     |  #  |  h  |  o  |  r  |  s  |  e  |
| --- | --- | --- | --- | --- | --- | --- |
| #   | 0   | 1   | 2   | 3   | 4   | 5   |
| r   | 1   | 1   | 2   | 2   | 3   | 4   |
| o   | 2   | 2   | 1   | 2   | 3   | 4   |
| s   | 3   | 3   | 2   | 2   | 2   | 3   |

## 代码
```
class Solution {
public:
    int minDistance(string word1, string word2) {
        //动态规划中相当程度的难题
        vector<vector<int>> dp(word2.size()+1,vector<int> (word1.size()+1,0));

        for(int i=0;i<=word1.size();i++)
        {
            dp[0][i]=i;
        }

        for(int i=0;i<=word2.size();i++)
        {
            dp[i][0]=i;
        }

        for(int i=1;i<=word2.size();i++)
        {
            for(int j=1;j<=word1.size();j++)
            {
                if(word2[i-1]==word1[j-1])//末尾字母相同则只取决于前面子串情况
                    dp[i][j]=dp[i-1][j-1];
                else
                    dp[i][j]=min(dp[i-1][j-1],min(dp[i][j-1],dp[i-1][j]))+1;//分别是替换、删除与插入
            }
        } 

        return dp[word2.size()][word1.size()];

    }
};
```
  
时间复杂度O($n^2$)
空间复杂度O($n^2$)
