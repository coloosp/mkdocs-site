# 移动0
[移动零](https://leetcode.cn/problems/move-zeroes/description/?envType=study-plan-v2&envId=top-100-liked)

注意题目所说要保持相对顺序，所以类似快排的思想是行不通的
使用快慢指针解决，我觉得我这个代码比题解还厉害，哈哈，这个代码不需要交换元素，也不需要提前统计0的个数
```
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int slow=0;//慢指针
        for(int fast=0;fast<nums.size();fast++)
        {
            if(nums[fast]!=0)
                nums[slow++]=nums[fast];//不等于0就复制
        }
        while(slow<nums.size())
            nums[slow++]=0; //fast已经将所有非零复制到前面了，所以这里直接把后面填为0即可
    }
};
```
