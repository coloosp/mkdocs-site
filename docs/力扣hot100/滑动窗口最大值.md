## 朴素暴力
没什么好多说的，纯暴力模拟
```
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        int left=0;
        int right=k-1;
        int maxnum=INT_MIN;
        vector <int> result;
        for(int i=0;i<k;i++){
            maxnum=max(nums[i],maxnum);
        }//初始化最大值
        
        while(right <nums.size()-1){
            result.push_back(maxnum);
            
            if(nums[left++]==maxnum)
                {
                    maxnum=INT_MIN;
                    for(int i=left;i<=right;i++) //如果弹出的是最大值就要重新检查
                        maxnum=max(maxnum,nums[i]);
                }
            
            if(nums[++right]> maxnum) //遇到新的最大值就更新
                maxnum=nums[right];

        }
        result.push_back(maxnum);
        
        return result;





    }
};
```
这段代码最好可以达到O(n)
不过最坏也可能是O(n²)

## 单调队列
### 解析
因为这道题是单调队列的模板题，所以大家可以深刻记忆这道题的形式
其实我在力扣上看到一个单调队列的笑话，我给大家翻译翻译。公司每年都会招一个新员工，我们公司裁员很严重，一旦新员工能力大于老员工，那把所有老员工全部裁员。然后检查是否有大于35岁的老员工，然后把它们裁员，剩下的一批要么是刚才招进来的新员工，要么是没被新员工KO的老员工。

**前端淘汰过时的，后端淘汰没用的**

单调队列一定要单调！这怎么做到呢？我们会把最大值保留在队首。新值会从队列尾部进入，然后弹出所有小于它的值。这样就保证我们的队列是从队首到队尾依次递减，因为我们需要完成从队尾弹出与队首弹出的两种操作，所以需要双端队列来进行操作
### 代码
```
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        //1、使用单调队列来解决这道题，单调队列里面需要存放元素下标
        //2、单调队列在遇见新元素时，弹出队列中所有小于它的元素的下标,再加入该元素
        //3、接下来如果我们已经形成窗口了，我们需要这样做：1.检查队首是否已经小于左窗口，如果是直接弹出 2.加入左窗口值
        vector <int> reuslt;
        deque<int> q;
        for(int right=0;right <nums.size();right++){
            while(!q.empty() && nums[right] >nums[q.back()])
                q.pop_back(); //从队尾依次弹出
        q.push_back(right);

        int left=right-k+1;
        if(q.front() <left)
            q.pop_front();//从队首弹出

        if(left>=0)//已经形成窗口，加入队首元素
            reuslt.push_back(nums[q.front()]);

        }

        return reuslt;

    }
};
```
## 双端队列
在C++中的双端队列
C++ 中的双端队列（deque，全称 double-ended queue） 是 STL 中功能更灵活的序列式容器，兼具「队列（FIFO）」和「栈（LIFO）」的特性
头文件 `#include <deque>`
定义 `deque<int> dq;`

|       操作方法        |              功能说明              | 时间复杂度 |            示例            |
| -------------------- | ---------------------------------- | ---------- | ------------------------- |
| `dq.push_back(val)`  | 队尾入队（同 queue::push）          | O(1)       | `dq.push_back(1);`        |
| `dq.push_front(val)` | 队首入队（queue 无此操作）          | O(1)       | `dq.push_front(2);`       |
| `dq.pop_back()`      | 队尾出队（无返回值）                | O(1)       | `dq.pop_back();`          |
| `dq.pop_front()`     | 队首出队（同 queue::pop）           | O(1)       | `dq.pop_front();`         |
| `dq.front()`         | 访问队首元素（同 queue）            | O(1)       | `int x = dq.front();`     |
| `dq.back()`          | 访问队尾元素（同 queue）            | O(1)       | `int y = dq.back();`      |
| `dq[i]` / `dq.at(i)` | 随机访问第i个元素（queue 无此操作） | O(1)       | `int z = dq[2];`          |
| `dq.empty()`         | 判断是否为空                        | O(1)       | `if (dq.empty()) return;` |
| `dq.size()`          | 返回元素个数                        | O(1)       | `int len = dq.size();`    |
| `dq.clear()`         | 清空所有元素                        | O(n)       | `dq.clear();`             |
