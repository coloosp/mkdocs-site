# 跳跃游戏
[跳跃游戏](https://leetcode.cn/problems/jump-game/description/?envType=study-plan-v2&envId=top-100-liked)

只要能跳就放心大胆跳就行

## 解析
这里我们的思路是反向思考，即将初始目标设置为终点nums.size()-1

向前检查有没有能跳到终点的点，如果有就更换目标，直到我们遇到起始点，如果起始点可以跳到目标点说明可以从起始点跳跃到终点

不知道各位自己想没想出来这个思路

贪心问题没有固定格式确实麻烦一点

我们会将这个目标不断贪心地向前移动，只要我们能够移动就不会考虑更多

## 代码

```
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int target=nums.size()-1; //目标位置
        for(int i=nums.size()-2;i>=0;i--)
        {
            if( i+nums[i] >=target) //如果这个位置可以跳我们就可以更换目标了
                target=i;
        }

        if(target==0) //这说明起始点可以跳跃都目标点
            return true;
        return false;
    }
};
```

时间复杂度O(n)
空间复杂度O(1)
