# 有效的括号
[有效的括号](https://leetcode.cn/problems/valid-parentheses/description/?envType=study-plan-v2&envId=top-100-liked)

经典老朋友应该也是数据结构的常见用例，各位只要把各种情况想清楚就没什么问题了

1. "[()]"正确
2. "()"正确
3. "(()" 错误，右括号少一个
4. "())"错误，左括号少一个
5. "[)"错误，括号不匹配

如果仅仅是一种括号也可以使用前缀和来完成

```
class Solution {
public:
    bool isValid(string s) {
        // 经典栈题，我们的好朋友呢
        stack<char> st;
        for (char c : s) {
            //出现右括号的情况
            if (c == '(' || c == '[' || c == '{'){
                st.push(c);
                continue;
            }
            //接下来是出现左括号的情况了
            
            if(st.empty()) 
                return false;
            
            char topchar=st.top();
            if (c == ')') {
                if (topchar != '(')
                    return false;

                st.pop();
            }
            if (c == ']') {
                if (topchar != '[')
                    return false;

                st.pop();
            }

            if (c == '}'){
                if (topchar != '{')
                    return false;

            st.pop();
            }
        }

        //也可能弹出完还有遗留的左括号
        if (!st.empty())
            return false;

        return true;
    }
};
```

时间复杂度O(n)
空间复杂度O(n)
