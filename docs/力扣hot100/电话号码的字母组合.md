# 电话号码的字母组合

[电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/?envType=study-plan-v2&envId=top-100-liked)

居然是字符串？作者一点都不喜欢字符串的题，不过这道题的关键不在于字符串，比如digits为"23"说明我们要收集2和3位置的数组，这种情况只需要写两个for循环即可

我们使用递归实现digits.size()个for循环，并且使用回溯来恢复每一个的收集路径，这就是回溯的本质了

## 解析
各位可能不知道我们的搜索数组nums之类的
实际上应该是根据图片中的信息自己构造

`string letterMap[10]={"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};`

回溯递归三步：
1. 确定参数：letterMap作为全局变量，传入digits与index
2. 确定终止条件：当路径s_path.size()==digts.size() 说明这条路上收集满了
3. 单层逻辑：确定在letteMap中当前的下标number=digits[index],遍历其中的字符数组

## 代码
```
class Solution {
public:
    vector<string> result;
    string s_path;//收集路径
    string letterMap[10]={"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"}; //搜索数组

    void backtracking(string digits,int index)
    {
        if(s_path.size()==digits.size()) //说明已经收集满
        {
            result.push_back(s_path);
            return ;
        }

            int number=digits[index]-'0';//强制转换为整型
            for(char c: letterMap[number])
            {
                s_path+=c;
                backtracking(digits,index+1);//进入下一层indedx+1
                s_path.pop_back();
            }
    }

    vector<string> letterCombinations(string digits) {
        backtracking(digits,0);
        
        return result;
    }
};
```
