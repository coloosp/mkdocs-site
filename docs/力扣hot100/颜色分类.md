## 常规做法
### 解析
我尽量说明为什么会这样做
首先我们要实现一个三指针
red:在red指针之前的值都是红色(0)
blue:在blue之后的值都是蓝色(2)
work:工作指针，从第一个非红色开始工作

对于work在循环work<=blue中遵循以下模式
* 当指向红色，与red对应值交换swap(nums[red],nums[work])然后让red和work同时进一
* 当指向白色，work进一，不做其他动作
* 当指向蓝色，与blue对应值交换，swap(nums[work],nums[blue]) blue--注意work不能前进，因为交换来的值也可能是蓝色的

作者确实只记得有这一套流程，解释思路恐怕作者讲不明白吧

### 代码
```
class Solution {
public:
    void sortColors(vector<int>& nums) {
        //本题要使用三指针进行排列
        //存在三个指针，red，blue，工作指针
        int red=0;
        int blue=nums.size()-1;
        int work=0;
        //red指针保证在它之前都为红，blue指针保证在它之后都为蓝
        for(int i=0;i<nums.size();i++)
        {
            if(nums[i]==0)
            {
                red++;
                work++;
            }
            else
                break;
        }

        for(int i=nums.size()-1;i>=0;i--)
        {
            if(nums[i]==2)
            {
                blue--;
            }
            else
                break;
        }

        while(work<=blue)
        {
            if(nums[work]==0) //红
            {
                swap(nums[red],nums[work]);
                red++;
                work++;
            }
            else if(nums[work]==1) //白
            {
                work++;
            }
            else //蓝
            {
                swap(nums[work],nums[blue]);
                blue--;
            }
        }


    }
};
```

时间复杂度O(n)
空间复杂度O(1)

### 等通知做法
类似于计数排序，提前知道红白蓝三种颜色然后重新排序
可以实现，但hr会让你回去等通知
```
class Solution {
public:
    void sortColors(vector<int>& nums) {
        //大家知道有个东西叫计数排序吗
        //小朋友们不要随便学，会被回家等通知的
        int red=0;
        int white=0;
        int blue=0;
        for(int i=0;i<nums.size();i++)
        {
            if(nums[i]==0)
                red++;
            else if(nums[i]==1)
                white++;
            else
                blue++;
        }
        int k=0;
        while(red--)
            nums[k++]=0;
        while(white--)
            nums[k++]=1;
        while(blue--)
            nums[k++]=2;

        return ;
    }
};
```

时间复杂度O(n)
空间复杂度O(1)
