# 排序链表
[排序链表](https://leetcode.cn/problems/sort-list/?envType=study-plan-v2&envId=top-100-liked)

应该是我做过的经典题目，可惜老方法会超时

## 经典模拟
不过会超时
```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        //链表经典题目，使用min和r来进行尾插法实现
        //使用p作为工作节点遍历链表的前驱节点
        ListNode *dummyhead=new ListNode;
        dummyhead->next=head;
        ListNode* min=dummyhead;//记录最小节点的前驱节点
        ListNode* r=dummyhead;//尾指针
        ListNode* p=head;//工作指针
        ListNode* t;//暂存指针
       while(r->next!=nullptr){
        p=r->next;
        min=r;
         while(p->next!=nullptr)
        {
            if(p->next->val < min->next->val)
                min=p;
            p=p->next;
        }
        t=min->next;
        min->next=t->next;//取出最小值点
        t->next=r->next;
        r->next=t;
        r=t;
       }
       ListNode *result=dummyhead->next;
       delete dummyhead;
       return result;
    }
};
```
时间复杂度O(n²)
空间复杂度O(1)

各位可以试着使用数组的思维诸如归并或者快排来实现，但这样并不符合题意，不建议这样做

## 归并排序
### 解析
各位如果没有学习过归并排序可能会看不懂后面的代码，如果各位之前学习过数据结构，那对于数组的归并排序应该了解清楚了
我们来做一个简单回顾吧
归并排序分为两个模块（我们这里三个）
* 归并模块用来把两个本身有序的数组合并为一个有序的数组
* 排序模块用来将一个无序的数组变得有序（应该是这个功能）  

因为归并排序使用了递归的思维，看起来并不直观，不过思路和快速排序的互通的，记住固定格式就不算难
### 代码
因为这里是处理链表所以会多出一个找到中间点的模块，你应该没忘记我们会使用快慢指针实现吧
代码如下：
```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* middleNode(ListNode* node) {
        ListNode* slow = node;
        ListNode* fast = node;
        ListNode* pre = node;
        while (fast && fast->next) // 利用快慢指针求解中间点
        {
            pre = slow; // 更新为slow的前驱
            slow = slow->next;
            fast = fast->next->next;
        }
        pre->next = nullptr; // 从中间断开分为两个链表
        return slow;         // 返回链表中点
    }

    // 归并两链表
    ListNode* mergelist(ListNode* list1, ListNode* list2) {
        ListNode dummyhead; // 虚拟头结点,栈区不会导致内存泄漏
        ListNode* r = &dummyhead;
        while (list1 && list2) {
            if (list1->val < list2->val) {
                r->next = list1;
                r = list1;
                list1=list1->next;
            } else {
                r->next = list2;
                r = list2;
                list2=list2->next;
            }
        }
        r->next = list1 ? list1 : list2; // 连接后续链表
        return dummyhead.next;
    }

    ListNode* sortList(ListNode* head) {
        // 如果分割的链表只有一个或者不存在
        if (head == nullptr || head->next == nullptr) {
            return head;
        }

        ListNode* head1 = middleNode(head); // 分割为两个子表
        head = sortList(head);              // 排序左子表
        head1 = sortList(head1);            // 排序右子表

        return mergelist(head, head1);
    }
};
```
时间复杂度O(nlogn)
空间复杂度O(logn) 这里是递归开销，注意如果是数组排序一般是O(n)
