# 三数之和
[三数之和](https://leetcode.cn/problems/3sum/?envType=study-plan-v2&envId=top-100-liked)
使用了双指针来实现
看起来只是两数之和的升级版，但是去重过程有点复杂其实，如果你要独立完成代码，我需要提醒的是，我们需要先锚定一个点，然后再使用双指针。
更重要的是，一定要先加入组合再进行去重操作，比如像[0,0,0]这种数组，如果先去重就会错误
```
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        //1、首先给数组排序
        //2、在后续剪枝和去重的原则都是先加入再去重
        //3、思路是先锚定一个数字，剩下两个数字使用双指针检查
        vector <vector<int>> result;
        sort(nums.begin(),nums.end());

        for(int i=0;i<nums.size();i++){
            if(nums[i]>0)//大于零时不可能在有正确的组合了
                continue;   
            if(i>0 && nums[i]==nums[i-1]) //这个数组已经检查一遍了
                continue;
            int left=i+1;
            int right=nums.size()-1;
            while(left<right){
                if( nums[left]+nums[right]+nums[i] >0) 
                    right--;
                else if(nums[left]+nums[right]+nums[i] <0) 
                    left++;
                else{
                    result.push_back({nums[i],nums[left],nums[right]});
                    while(left<right && nums[left]==nums[left+1]) left++;
                    while(left<right && nums[right]==nums[right-1]) right--;

                    //这里还需要滑动一次的原因是，上面的最后一次滑动去重是把两个指针滑动到了相同组合的边缘，再滑动一次才会到达新组合
                    left++;
                    right--;
                } 



            }



        }
        return result;
            }
};
```
时间复杂度O(n²)
空间复杂度O(1)
各位可以自行举例来思考为什么还需要再滑动一次
