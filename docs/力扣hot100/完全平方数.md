## 解析
对于任意一个整数，我们一定会进行以下的操作，将其拆分为一个完全平方数+剩下的整数，然后对剩下的整数进行再拆分

我们知道任意一个完全平方数需要的完全平方数个数一定是1

其实操作就很简单了，对于一个整数k，我们尝试不大于k的完全平方数1,4,9..进行拆分 得到 j* j与(k-j*j)
假设dp[i]表示i需要拆分的个数，dp[k]=min(dp[k],1+dp[k-j*j]) 
dp[k-j*j]哪来？答案是之前算过了！这也是动态规划的特性，我们会计算之前的最优子结构

1. **确定dp数组与下标的含义**：dp[i]表示对于整数i的最少完全平方数数量
2. **确定递推公式**：dp[i]表示对于整数i的最少完全平方数数量
3. **dp数组初始化**：dp[0]=0,其他初始化为INT_MAX表示未探索
4. **确定遍历顺序**：从1开始向n遍历，并从平方数1开始遍历
5. **举例推导dp数组**
比如5
dp[0]=0
dp[1]=1+dp[0]=1
dp[2]=1+dp[1]=2
dp[3]=1+dp[2]=3
dp[4]=min(1+dp[0],1+dp[3])=1 //拆分为1,3或者4,0
dp[5]=min(1+dp[4],1+dp[1]) =2 //拆分为1,4或者4,1

## 代码
```
class Solution {
public:
    int numSquares(int n) {
        vector<int>dp (n+1,INT_MAX); //dp[i]表示对于整数i的最少完全平方数数量
        dp[0]=0; //0不需要任何完全平方数
        for(int i=1;i<=n;i++)
        {
            for(int j=1;j*j<=i;j++)//将其拆分为一个完全平方数+剩余整数
                {
                    dp[i]=min(dp[i],dp[i-j*j]+1);//可以把i拆分为j*j +i-j*j,dp[i-j*j]在前面已经被计算过了
                }
        }

        return dp[n];
    }
};
```

时间复杂度O($n^2$)
空间复杂度O(n)
