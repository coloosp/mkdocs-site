# 只出现一次的数字

[只出现一次的数字](https://leetcode.cn/problems/single-number/description/?envType=study-plan-v2&envId=top-100-liked)

居然来到这里了吗？技巧章节更像是脑筋急转弯，里面的很多技巧很可能想不到，即使是简单题也不是那么容易  
## 解析
本题只有两个字——**异或**

没什么好说的，大家看看异或的特性就知道了

|             公式            |               说明               |      示例（十进制）       |           二进制验证（以8位为例）           |
| --------------------------- | -------------------------------- | ----------------------- | ----------------------------------------- |
| `a ^ a = 0`                 | 相同数异或=0                      | `5 ^ 5 = 0`             | `00000101 ^ 00000101 = 00000000`          |
| `a ^ 0 = a`                 | 任何数异或0=自身                  | `5 ^ 0 = 5`             | `00000101 ^ 00000000 = 00000101`          |
| `a ^ b = b ^ a`             | 交换律                           | `5 ^ 3 = 3 ^ 5 = 6`     | `00000101^00000011=00000110`              |
| `(a ^ b) ^ c = a ^ (b ^ c)` | 结合律                           | `(5^3)^2 = 5^(3^2) = 1` | `(6)^2=4；5^(1)=4`（注：8位二进制验证一致） |
| `a ^ b ^ b = a`             | 异或同一个数两次=自身（交换数核心） | `5 ^ 3 ^ 3 = 5`         | `00000101^(00000011^00000011)=5^0=5`      |  
  
真的没什么好说的了，因为只要出现两次的数就会被异或变成0，而0和其他数的异或又是其他数本身，想不到就没法做，因为本题只允许使用常数空间，并不能使用哈希表，就这样吧

然后值得一提的是，异或也可以用来实现不使用中间值进行两个数的交换

## 代码
```
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        //无言投下...
        //答案是异或
        int num=nums[0];
        for(int i=1;i<nums.size();i++)
        {
            num=num^nums[i];
        }

        return num;
    }
};
```
时间复杂度O(n)
空间复杂度O(1)
