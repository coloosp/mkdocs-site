# 多数元素
[多数元素](https://leetcode.cn/problems/majority-element/description/?envType=study-plan-v2&envId=top-100-liked)
当然各位也可以使用哈希表来做，我就不展示了

介绍一种算法——摩尔投票法，这是作者在学习数据结构时学会的算法
## 解析
给大家讲个故事吧
话说力扣拳击赛正在火热举办中，我们的选手根据自身数字划分为不同阵营，选手们实力相当，我们选取第一位选手nums[0]作为擂主，如果接下来遇到相同阵营，就可以邀为同道，让己方阵营积分+1，如果遇到敌方就需要消耗一名选手，让己方阵营积分-1，如果积分变为-1说明该方选手消耗光了，更新为当前nums[i]作为擂主，重置积分为1。活到最后的才是赢家！

## 代码
```
class Solution {
public:
    int majorityElement(vector<int>& nums) {
        //使用了著名的摩尔投票法
        //唯有死斗方能存活
        int curnum=nums[0]; //初始擂主
        int count=1;
        for(int i=1;i<nums.size();i++)
        {
            if(nums[i]!=curnum)//扣除积分！
                count--; 
            else
                count++; //邀为同道！
            if(count <0)
            {
                curnum=nums[i]; //更换擂主
                count=1; //重置积分
            }
        }

        return curnum; //因为题目说明必然存在一个多数元素
    }
};
```
