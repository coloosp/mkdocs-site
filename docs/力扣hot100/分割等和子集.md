# 分割等和子集
[分割等和子集](https://leetcode.cn/problems/partition-equal-subset-sum/?envType=study-plan-v2&envId=top-100-liked)

本质就是01背包问题
假设总大小为ans
你可以理解对于这些元素是否可以装满一个一半大小ans/2的背包

并且要注意每一个元素只能使用一次(这是和完全背包的根本区别)

这里也很像前面的完全平方和
比如如何判断一个j容量的背包是否被装满
对于nums[i]来说，我们划分为两部分 nums[i]、 j-nums[i]
j-nums[i]是否能够装满前面已经判断过了！

总之让我们动态规划五步分析清楚吧！

对于各位来说一般二维dp会比一维dp好写，但是这道题个人体感一维dp反而冗余了
1. **确定dp数组与下标的含义**：bool类型 dp[i][j]对于遍历到第i个物品时，容量j的背包是否能够装满
2. **确定递推公式**:首先继承上一次的dp数组dp[i][j]=dp[i-1][j] 如果j<nums[i-1]则不进行后续操作。否则判断对于j-nums[i-1]的背包是否装满，即dp[j-num[i-1]]==true,如果是则说明dp[i][j]能够装满
3. **dp数组初始化**:需要把容量0的位置设置为true，这里无论如何都能被装满
4. **确定遍历顺序** :因为是组合数，所以是先物品再背包
5. **举例推导dp数组**
例如[2,2,1,1] ans=6 half=3

|      |  0  |  1  |  2  |  3  |
| ---- | --- | --- | --- | --- |
| null | √   | x   | x   | x   |
| 2    | √   | x   | √   | x   |
| 2    | √   | x   | √   | x   |
| 1    | √   | √   | √   | √   |
| 1    | √   | √   | √   | √   |

其实一维滚动数组会简单一些，这是经典模板题，各位根据自身情况决定吧

## 代码
### 二维背包
```
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        //简单的背包问题，其实是01背包的变体
        //核心思路是将任意元素装入背包中，观察是否能装满容量为总和一半的背包

        int ans=0;
        for(int i=0;i<nums.size();i++)
        {
            ans+=nums[i];
        }
        if(ans%2!=0) //对于奇数总和是不可能被分割的
            return false;

        int half=ans/2;
        vector <vector<bool>> dp(nums.size()+1,vector<bool>(half+1,false));
        
        for(int i=0;i<=nums.size();i++)
        {
            dp[i][0]=true;
        }

        //组合数的话我们需要先遍历物品再遍历背包
        for(int i=1;i<=nums.size();i++)
        {
            for(int j=1;j<=half;j++)
            {
                dp[i][j]=dp[i-1][j];
                if(j <nums[i-1]){
                    continue;
                }

                if(dp[i-1][j-nums[i-1]]==true)
                    dp[i][j]=true;
            }
        }
        
        return dp[nums.size()][half];
    
    }
};
```

时间复杂度O($n^2$)
空间复杂度O($n^2$)

### 一维滚动背包
各位要尝试的话一定要在遍历背包时反向遍历，否则会变成完全背包的形式
```
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        //简单的背包问题，其实是01背包的变体
        //核心思路是将任意元素装入背包中，观察是否能装满容量为总和一半的背包

        int ans=0;
        for(int i=0;i<nums.size();i++)
        {
            ans+=nums[i];
        }
        if(ans%2!=0) //对于奇数总和是不可能被分割的
            return false;

        int half=ans/2;
        vector <vector<bool>> dp(nums.size()+1,vector<bool>(half+1,false));
        
        for(int i=0;i<=nums.size();i++)
        {
            dp[i][0]=true;
        }

        //组合数的话我们需要先遍历物品再遍历背包
        for(int i=1;i<=nums.size();i++)
        {
            for(int j=1;j<=half;j++)
            {
                dp[i][j]=dp[i-1][j];
                if(j <nums[i-1]){
                    continue;
                }

                if(dp[i-1][j-nums[i-1]]==true)
                    dp[i][j]=true;
            }
        }
        
        return dp[nums.size()][half];
    
    }
};
```

时间复杂度O($n^2$)
空间复杂度O(n)
