# 盛最多水的容器
[盛最多水的容器](https://leetcode.cn/problems/container-with-most-water/?envType=study-plan-v2&envId=top-100-liked)
两个for循环一定会超时的，不然这道题也不会是中等题了

使用双指针来完成，核心思想是在收缩探索时，我们只会移动长度更小的那一条，这样就保证了水量尽可能大。

其实第一次想出这种解法不算容易，但知道这种类型的题就不难了

```
class Solution {
public:
    int maxArea(vector<int>& height) {
        int left=0;
        int right=height.size()-1;
        int result=0;
        while(left<right){
            int area=min(height[left],height[right])*(right-left); //计算水量
            result=max(result,area);
            if(height[left]<height[right]) //移动更小的棍子
                left++;
            else
                right--;
        }
        return result;

    }
};
```
