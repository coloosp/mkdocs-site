# 两两交换链表中的节点
[两两交换链表中的节点](https://leetcode.cn/problems/swap-nodes-in-pairs/?envType=study-plan-v2&envId=top-100-liked)

请各位牢记头插法尾插法双指针这三个思维。这是我们解决所有链表问题的强大工具~(￣▽￣)~*

作者之前在没有系统学习过链表时也做过这道题，被折磨地很惨www

```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* swapPairs(ListNode* head) {
        // 相信学习过头插法的各位知道怎么去交换吧
        // 我们会使用一个前驱节点来管理两个后继节点的交换
        // 处理的其他小细节还不少，我们仔细完成吧！
        ListNode* dummyHead = new ListNode; // 依旧头结点
        dummyHead->next = head;

        ListNode* pre = dummyHead;

        // 如果前驱节点后面没有节点或者只有一个节点，实际上是不需要处理的
        while (pre->next != nullptr && pre->next->next != nullptr) {
            ListNode* p1 = pre->next;
            ListNode* p2 = pre->next->next;
            pre->next = p2->next;

            // 头插法逻辑
            p1->next = pre->next;
            pre->next = p1;

            p2->next = pre->next;
            pre->next = p2;

            pre = p1; // 更新前驱节点
        }
        ListNode* result = dummyHead->next;
        delete dummyHead;
        return result;
    }
};
```

时间复杂度 O(n)
空间复杂度O(1)
