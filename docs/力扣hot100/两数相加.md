# 两数相加
[两数相加](https://leetcode.cn/problems/add-two-numbers/?envType=study-plan-v2&envId=top-100-liked)

不知道各位有没有写过大数加法，这道题应该算是大数加法的链表版
细节挺多，总之我是费了一些功夫来完成这道题的，这个版本的代码应该更容易看懂了

```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
        //因为测试案例会在100位数，会远大于int的范围，所以别想收集下来计算
        //在计算前我们可能需要先判断哪一个数长度更长
        //我们会使用flag来标记是否有进位，反正进位只有0和1
        //计算时使用对10的取模操作
        //我们会创建带头结点的新链表来返回，这样处理更方便

        ListNode *Head =new ListNode;
        ListNode *r=Head;//尾节点

        int flag=0;//进位标志

        ListNode *p1=l1;
        ListNode *p2=l2;

        while(p1!=nullptr || p2!=nullptr || flag!=0){
            int val1= (p1!=nullptr) ? p1->val:0;
            int val2= (p2!=nullptr) ? p2->val:0;

            int sum=val1+val2+flag;
            int curval=sum%10;//当前节点值
            flag =sum/10;

            ListNode *t=new ListNode;
            t->val=curval;
            r->next=t;
            r=t;

            if(p1!=nullptr) p1=p1->next;
            if(p2!=nullptr) p2=p2->next;
        }
        r->next=nullptr; //置空
        
        ListNode *result=Head->next;
        delete Head;
        return result;


    }
};
```

时间复杂度O(n)
空间复杂度O(n)之前试过将更大的链表作为基础链表存储结果，但是效果不是很好，还是建议自己搓一个链表出来
