# 合并区间
[合并区间](https://leetcode.cn/problems/merge-intervals/?envType=study-plan-v2&envId=top-100-liked)

这里是排序+模拟的思路，虽然不是最优秀的，但好在简单易想，开袋即食说是

```
class Solution {
public:
    static bool cmp(const vector<int>&a,const vector<int>&b){//按照左端点进行排序
        return a[0]<b[0];
    }

    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        //首先对数组进行排序，这样做的原因我们在后面逐个加入时就可以只考虑右区间问题了
        // 使用模拟的方式完成，使用一个栈了存储结果。如果栈空则压入区间，否则会检查压入元素和栈顶的关系，选择如何合并两栈
        sort(intervals.begin(),intervals.end(),cmp);
        vector<vector<int>> result;

        for (vector<int> x : intervals) {
            if (result.empty()) {
                result.push_back(x);
            } else {
                vector<int> top = result.back();
                int left_top = top[0];
                int right_top = top[1];
                int left_back = x[0];
                int right_back = x[1];
                if (left_back >= left_top && left_back <= right_top &&
                    right_back >= left_top && right_back <= right_top) {
                    continue;
                } else if (left_back >= left_top && left_back <= right_top &&
                           right_back > right_top) {
                    result.pop_back();
                    result.push_back({left_top, right_back});
                } else if (right_back >= left_top && right_back <= right_top &&
                           left_back < left_top) {
                    result.pop_back();
                    result.push_back({left_back, right_top});
                } else if (left_back < left_top && right_back > right_top) {
                    result.pop_back();
                    result.push_back({left_back, right_back});
                } else {
                    result.push_back({left_back, right_back});
                }
            }
        }






        return result;
    }
};
```

时间复杂度O(nlogn) 因为我们使用了快速排序
空间复杂度O(n) 这应该是快速排序的函数栈开销
作者虽然写了一大串，但其实都是垃圾逻辑，应该也是可以轻松完成的
