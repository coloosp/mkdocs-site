# 寻找选择排序数组中的最小值
[寻找选择排序数组中的最小值](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/?envType=study-plan-v2&envId=top-100-liked)

反而是弱化版的上一题，思路完全一致，各位学习完上一题就可以轻松实现这道题了

```
class Solution {
public:
    int findMin(vector<int>& nums) {
        //居然把这道题放在后面
        //有了前面一题的思路这里很好解决了
        int last=nums[nums.size()-1];
        int low=0;
        int high=nums.size()-1;

        while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>last) //还在第一段
                low=mid+1;
            else                //处于第二段了
                high=mid-1;
        }

        return nums[low]; //得到的low就是最小值下标
    }
};
```
