# 反转链表
[反转链表](https://leetcode.cn/problems/reverse-linked-list/description/?envType=study-plan-v2&envId=top-100-liked)

也是链表里面一定会学习到的头插法，不过注意这里的head并不是我们说的头结点，而是链表的第一个元素

作者喜欢将工作指针和暂存指针分离，工作指针只是做锚定和移动的工作

总之学习过数据结构的各位一定也能很轻松写出这个

```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        //经典头插法，但是这里head实际上指首节点
        if(head==nullptr)
            return head;
        ListNode*p=head->next;//工作节点
        ListNode*t;//暂存节点
        head->next=nullptr;
        while(p!=nullptr){
            t=p;
            p=p->next;
            t->next=head;
            head=t;
        }
        return head;


    }
};
```

时间复杂度O(n)
空间复杂度O(1)
