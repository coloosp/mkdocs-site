# 两数之和
[两数之和](https://leetcode.cn/problems/two-sum/?envType=study-plan-v2&envId=top-100-liked)
梦开始的地方
## 暴力算法
```
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector <int> result(2);
        for(int i=0;i<nums.size();i++){
            for(int j=i+1;j<nums.size();j++){
                if(nums[i]+nums[j]==target){
                    result[0]=i;
                    result[1]=j;
                    return result;
                }
            }
        }

        return {};

    }
};
```

朴实无华
时间复杂度O(n²)
空间复杂度O(1)

## 哈希表
```
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map <int,int> hashmap; //值键对形式的哈希表
        for(int i=0;i<nums.size();i++){
            auto it=hashmap.find(target-nums[i]);//查找是否存在
            if(it!=hashmap.end()){
                return {i,it->second};
            }
                hashmap.insert({nums[i],i});//加入值键对
        }
        return{};
    }
};
```

需要一些STL的知识，比如因为unordered_map是无序表，平均增删查都是O（1）。

时间复杂度O(n)
空间复杂度O(1)
