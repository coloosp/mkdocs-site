# 子集
[子集](https://leetcode.cn/problems/subsets/?envType=study-plan-v2&envId=top-100-liked)

## 解析
比起全排列这里的区别有二，第一，子集不需要满足path.size()==nums.size()只要是新的子集就可以加入
第二是要小心加入重复的子集，因此需要使用index来标记当前搜索下标，下一次只能往这更后搜索

回溯递归三步
1. 确定参数：nums与index
2. 当index等于nums.size()不再会继续搜索自动结束
3. 遍历nums并且向下一层函数传入i+1作为下标

## 代码
```
class Solution {
public:
    vector<vector<int>> result;
    vector<int> path;

    void backtracking(vector<int>& nums,int index)
    {
        result.push_back(path);//见到一个收集一个
        for(int i=index;i<nums.size();i++)
        {
            path.push_back(nums[i]); //在循环内收集应该是更合适的
            backtracking(nums,i+1); //下一次从i+1开始
            path.pop_back(); //恢复
        }
    }

    vector<vector<int>> subsets(vector<int>& nums) {
        backtracking(nums,0);

        return result;
    }
};
```
