# 搜索二维矩阵
[搜索二维矩阵](https://leetcode.cn/problems/search-a-2d-matrix/?envType=study-plan-v2&envId=top-100-liked)

看起来确实比它的好兄弟[搜索二维矩阵II](https://leetcode.cn/problems/search-a-2d-matrix-ii/description/?envType=study-plan-v2&envId=top-100-liked)简单一些

思路还是二分搜索，我们先搜索第一列的行确定target可能所在的行数，再二分搜索确定所在的列就可以完成搜索了，感觉属于简单题（

一点小技巧是，至少在我的写法里面，如果搜索不到数，low会比high高一位，这时候可以根据high来确定我们需要搜索的行，不过注意high可能为-1，这时说明搜索不到返回false即可

```
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        //应该是先按行二分，再按列二分
        //一个小知识是当搜索不到时high会比low小一位
        int m=matrix.size();
        int n=matrix[0].size();
        //开始行搜索
        int low=0;
        int high=m-1;
        while(low<=high)
        {
            int mid=(low+high)/2;
            if(matrix[mid][0]==target)
                return true;

            if(matrix[mid][0]>target)
                high=mid-1;
            else
                low=mid+1;
        }

        if(high<0) //特判防止超界
            return false;
        int row=high;//可能所在行位于high位置

        low=0;
        high=n-1;
        //开始列搜索
        while(low<=high)
        {
            int mid=(low+high)/2;
            if(matrix[row][mid]==target)
                return true;
            if(matrix[row][mid]>target)
                high=mid-1;
            else
                low=mid+1;
        }

        //全都搜索不到则不存在
        return false;
    }
};
```

时间复杂度O(logn)
空间复杂度O(1)
