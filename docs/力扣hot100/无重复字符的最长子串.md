# 无重复字符的最长子串
[无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/?envType=study-plan-v2&envId=top-100-liked)

## 暴力+STL
哈哈，我都没想到可以通过，力扣还是太温柔了
```
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int result=0;
        for(int i=0;i<s.size();i++){
            int len=0;
            unordered_set<char> st;//去重集合
            for(int j=i;j<s.size();j++){
                if(st.count(s[j])!=0)
                    break;
                st.insert(s[j]);
                len++;
            }
            result=max(result,len);

        }

        return result;
    }
};
```
时间复杂度O(n²)
空间复杂度O(n)
## 滑动窗口
滑动窗口也是双指针的一部分（可以这样说吗）
这里的核心思路是右窗口一直向前移动，左窗口当窗口内出现重复字符（这是因为右窗口移动导致的），左窗口要一直移动直到窗口符合条件，即不存在相同字符
```
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char> window; //滑动窗口
        int result=0;
        int left=0;
        for(int right=0;right<s.size();right++){
            while(window.count(s[right])!=0){ //包含重复元素
                window.erase(s[left]);
                left++;//缩小窗口
            }
            window.insert(s[right]); //加入最左侧元素
            result=max(result,right-left+1);
        }
        return result;
    }
};
```
时间复杂度O(n)
空间复杂度O(n)
