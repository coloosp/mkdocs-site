# 最长递增子序列
[最长递增子序列](https://leetcode.cn/problems/longest-increasing-subsequence/?envType=study-plan-v2&envId=top-100-liked)

这道题就和前面不一样了，这道题并非背包问题，但是也属于动态规划的部分。一定要清楚，我们的动态规划会存储最优子结构，我们试着拆分一下这个问题

比如对于这个例子nums = [10,9,2,5,3,7,101,18]
想问的是以18为结尾的递增子序列有哪些呢？我不知道全部，但我发现7应该在其中，我再试着询问，以7为结尾的递增子序列有哪些呢？答：7为结尾的递增子序列在之前已经搜索过了

实际上这是很有趣的现象，我们会在思考时自顶向下思考，而动态规划的实现却是自底向上的。

好的，下结论吧，对于nums[i]的最长递增子序列，只需要找到在i之前的结尾比nums[i]更小的最长递增子序列

我们先确定dp[i]表示以nums[i]结尾的最长递增子序列长度

并且因为任何数字至少可以把本身当做结尾，所以任何dp[i]初始都应该为1

1. **确定dp数组与下标的含义**：dp[i]表示以nums[i]结尾的最长递增子序列长度
2. **确定递推公式**dp[i]=max(dp[i],dp[j]+1) j在i前面且nums[j]<nums[i]
3. **dp数组初始化**:全部初始化为1
4. **确定遍历顺序** ：对于i需要从1开始遍历，j可以以任意顺序遍历
5. **举例推导dp数组**：

| 10  |  9  |  2  |  5  |  3  |  7  | 101 | 18  |
| --- | --- | --- | --- | --- | --- | --- | --- |
| 1   | 1   | 1   | 2   | 2   | 3   | 4   | 4   |

## 代码
```
class Solution {
public:
    int lengthOfLIS(vector<int>& nums) {
        vector<int> dp(nums.size(), 1);//dp[i]表示以nums[i]为止的最长递增子序列长度,因为本身也算所以至少为1
        int result = 1;
        for (int i = 1; i < nums.size(); i++) {
            for (int j = i-1; j >=0; j--) { //向前搜索，其实这里顺序不重要，反正要全部搜索一遍
                if (nums[i] > nums[j]) { //递增子序列一定要递增
                    dp[i] = max(dp[i], dp[j] + 1); //更新直到找到最长递增子序列
                }
            }

            result = max(result, dp[i]);//因为并非dp[nums.size()]为最大值，我们要及时更新
        }

        return result;
    }
};
```

时间复杂度O($n^2$)
空间复杂度O(n)
