# 在排序数组中查找元素的第一个和最后一个位置
[在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

应该算是经典的模板题，虽然看起来简单，但是很多边界问题值得我们认真思考

对于这种题应该做出一定的总结，我做的时候也错了几次

## 解析
* 进行偏向于high移动的二分查找，low为元素的左边界
```
while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>=target)
                high=mid-1;
            else
                low=mid+1;
        }
```

* 进行偏向于low移动的二分查找，high为元素的右边界
```
 while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>target)
                high=mid-1;
            else
                low=mid+1;
        }
```

还要注意的是及时判断low与high是否超界，以及边界上是否是我们需要的目标值

```
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        //使用两次特殊的二分搜索就可以确定,应该需要注意一些超界问题
        if(nums.size()==0)
            return {-1,-1};
        int low=0;
        int high=nums.size()-1;
        //确定数值的下边界
        while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>=target)
                high=mid-1;
            else
                low=mid+1;
        }

        if(low>nums.size()-1)//下边界超界
            return {-1,-1};

        if(nums[low]!=target)
            return {-1,-1};
        
        int left=low;
        low=0;
        high=nums.size()-1;
        //确定数值的上边界
         while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>target)
                high=mid-1;
            else
                low=mid+1;
        }

        if(high<0)//上边界超界
            return {-1,-1};

        if(nums[high]!=target)
            return {-1,-1};
        int right=high;

        return {left,right};
    }
};
```
