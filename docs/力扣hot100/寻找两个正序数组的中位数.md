# 寻找两个正序数组的中位数
[寻找两个正序数组的中间值](https://leetcode.cn/problems/median-of-two-sorted-arrays/?envType=study-plan-v2&envId=top-100-liked)

这是一道相当有价值的困难题，有着有趣的思路，边界处理，特例判定等，可以说是想出来困难做起来更困难，不过也是标准的模版题(作者在学习数据结构时也遇到了这道题)

## 解析
对于一个有序数组的中位数应该怎么算？如果长度是len的话，中位数的下标k=len/2,当然这还要区分len为偶数与奇数。也就是说**寻找中位数也就是寻找第k小的数字**

根据奇偶情况，如果为偶数的话应该是第len/2,len/2+1两个元素，奇数的话应该是len/2+1的位置

这是一个关于舍弃的故事，好像什么武侠小说或者游戏里面的桥段，舍弃宝物或者什么道具才能开启道路一样。核心思路是**要寻找第k个元素就舍弃前面k-1个元素**

当然我们不可能一次性舍弃k-1个元素，因为我们不知道中位数会在哪一个数组中，这时候转换思路，我们先舍弃k/2个元素，逐步逼近到k-1个，也就是k/2+k/4+...+1=k-1

为此我们可以这样做，先比较两数组前k/2个数组(如果有的话)，舍弃其中更小的一份，然后逐步逼近到总共舍弃k-1个

解释以下定义
* k：总共需要舍弃的元素个数
* index：当前数组的开始端下标
* step：步长，当前循环舍弃的元素个数  
  
核心思路就是这样的，两边拿出step个元素对拼，小的一方舍弃这些元素，index增加step个，直到k=1说明已经舍弃完了

但是这还没完，有很多问题我们没有解决，比如某一个数组在一次循环中拿不出step个元素，我们只能要求它舍弃到结尾，也就是index会移向nums.size()，这时候我们直接结束循环，说明中位数应该在另一个数组中，应该是index+k-1的位置

还有更坑的是，有可能一开始这些数组就是老赖，为空数组时，或者在最后一轮循环中刚好index=nums.size()而跳出循环时，我们需要在外部进行特判

## 代码
```
class Solution {
public:
    int binarysearch(vector<int>&nums1,vector<int>&nums2,int k)
    {
        int index1=0;
        int index2=0;
        while(k>1)
        {
            //边界判定
            if(index1 >= nums1.size())
                return nums2[index2 + k - 1];
            if(index2 >= nums2.size())
                return nums1[index1 + k - 1];

            int step=k/2;

            int pos1=min(index1+step-1,(int)nums1.size()-1);
            int pos2=min(index2+step-1,(int)nums2.size()-1);

            if(nums1[pos1] <nums2[pos2])
            {
                k-=(pos1-index1+1);
                index1=pos1+1;
            }
            else
            {
                k-=(pos2-index2+1);
                index2=pos2+1;
            }

        }

        //防空数组的特判
        if(index1 >= nums1.size())
            return nums2[index2];
        if(index2 >= nums2.size())
            return nums1[index1];

        return min(nums1[index1],nums2[index2]);
    }

    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        //理解是求中位数是求第k小的元素
        //这是关于舍弃的故事...
        int m=nums1.size();
        int n=nums2.size();
        if((m+n)%2==1){
            int k=(m+n)/2+1;
            return binarysearch(nums1,nums2,k);
        }
        else
        {
            int k=(m+n)/2;
            //防止精度损失
            int val1 = binarysearch(nums1, nums2, k);
            int val2 = binarysearch(nums1, nums2, k+1);
            double result = (double)(val1 + val2) / 2;
            return result;
        }

    }
};
```

时间复杂度O(log(min(m,n)))
空间复杂度O(1)

class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        int m=nums1.size();
        int n=nums2.size();
        vector <int> nums(m+n,0);

        int i=0;
        int j=0;
        int k=0;
        while(i<nums1.size() && j<nums2.size())
        {
            if(nums1[i]<nums2[j])
                nums[k++]=nums1[i++];
            else
                nums[k++]=nums2[j++];
        }

        while(i<nums1.size())
            nums[k++]=nums1[i++];
        while(j<nums2.size())
            nums[k++]=nums2[j++];

        if(nums.size()%2==0)
        {
            int half=nums.size()/2;
            return (nums[half]+nums[half-1])/2.0;
        }

        return nums[nums.size()/2];
    }
};
