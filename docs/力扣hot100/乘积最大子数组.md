# 乘积最大子数组
[乘积最大子数组](https://leetcode.cn/problems/maximum-product-subarray/?envType=study-plan-v2&envId=top-100-liked)

有趣的动态规划题，其实做之前可以考虑一下[最大子数组和](https://leetcode.cn/problems/maximum-subarray/description/)的动态规划写法

对于子数组和的话，dp[i]=max(nums[i],nums[i]+dp[i-1])对吧，核心思路是只要是正数都可以往里面塞

## 解析
但这个有一点不同的是，当前的最小值真的该被抛弃吗？可惜这里的例子不能很好体现
比如[-1,-2,-3-4]
正确答案应该是24，如果按照前面的规律dp[2]也就是对应-3的值应该是-3而不是-6，但是dp[3]的-4却需要-6的信息

已经呼之欲出了，给大家说明吧 **最小值和最大值同样重要，因为存在负数可以相互转换**

因为存在下一个值为负数，那我们的最小值就可能绝地翻盘(好中二)
所以思路就是使用两个dp数组来存储，一个是最大值maxdp一个是最小值mindp

1. **确定dp数组与下标的含义**：maxdp[i]表示需要包含nums[i]最大连续子数组乘积，mindp[i]表示需要包含nums[i]的最小连续子数组乘积
2. **确定递推公式**:因为两者随时可能转换，递推公式应该是`maxdp[i]=max(nums[i],max(maxdp[i-1]*nums[i],mindp[i-1]*nums[i] ));`
`mindp[i]=min(nums[i],min(maxdp[i-1]*nums[i],mindp[i-1]*nums[i]))` 
3. **dp数组初始化**:maxdp和mindp的第一个元素都应该是nums[0]
4. **确定遍历顺序** :从第二个元素开始遍历即可
5. **举例推导dp数组**
 nums = [2,3,-2,4]
` maxdp[0]=2`  `mindp[0]=2`
`maxdp[1]=max(3,2*3,2*3)=6`
`mindp[1]=min(3,2*3,2*3)=2`
`maxdp[2]=max(-2,-2*6，-2*2)=-2`
`mindp[2]=min(-2,-2*6,-2*2)=-12`
`maxdp[3]=max(4,-2*4,-12*4)=4`
`mindp[3]=min(4,-2*4,-12*4)=-48`

## 代码
```
class Solution {
public:
    int maxProduct(vector<int>& nums) {
        //这里因为存在负数的原因要使用两个dp数组
        vector <int> maxdp(nums.size());
        vector<int>  mindp(nums.size());
        int result=nums[0];
        maxdp[0]=nums[0];
        mindp[0]=nums[0];
        for(int i=1;i<nums.size();i++)
        {
            maxdp[i]=max(nums[i],max(maxdp[i-1]*nums[i],mindp[i-1]*nums[i]));
            mindp[i]=min(nums[i],min(maxdp[i-1]*nums[i],mindp[i-1]*nums[i]));
            result=max(maxdp[i],result);
        }

        return result;
    }
};
```

时间复杂度O(n)
空间复杂度O(n)
