# 最长公共子序列
[最长公共子序列](https://leetcode.cn/problems/longest-common-subsequence/description/?envType=study-plan-v2&envId=top-100-liked)

这道也是经典老题了，作者两年前比赛时就写过一模一样的题，不过思路和上一题其实很类似

## 解析
我们先来确定dp[i][j]是什么意思吧
`vector<vector<int>> dp(text2.size(),vector<int>(text1.size(),0))`
表示对于text2的[0,i)区间与text1[0,j)区间的公共子序列

我们举个例子来看看  
text1 = "abcde", text2 = "ace" 

|     |  #  |  a  |  b  |  c  |  d  |  e  |
| --- | --- | --- | --- | --- | --- | --- |
| #   | 0   | 0   | 0   | 0   | 0   | 0   |
| a   | 0   | 1   | 1   | 1   | 1   | 1   |
| c   | 0   | 1   | 1   | 2   | 2   | 2   |
| e   | 0   | 1   | 1   | 2   | 2   | 3   |

我们思考一下其中的子串对比 比如"abc"与"ac"
我们对比到这里时，我们知道两个结尾的c相同，但是"ab"与"a"呢？也就是dp[1][2],答案是前面已经算过了(这句话我不知道说了多少遍)
也就是说当text1[j-1]与text2[i-1]相同时，我们去掉这两个字母，比较剩下子串的最长公共子序列
也就是吧"abc"="ab"+'c'
"ac"='a'+'c'

说明两个字母相同时dp[i][[j]=dp[i-1][j-1]+1

怎么样，和前面的回文子串是不是很类似
我们之前是这样操作的，
"abcba"='a'+"bcb"+'a'

**不过要是两个字母不相同呢？**
比如"ace"与"abcd"
其实也不难想 因为在比较这个之前我们就已经知道了"ac"与"abcd"的最长公共子序列 以及"ace"与"abc"的最长公共子序列了
反正e和d不会贡献公共子序列个数，所以不妨去掉任意一个比较并取最大值即可

让我们五步分析
1. **确定dp数组与下标的含义**：dp[i][j]表示对于text2的[0,i)区间与text1[0,j)区间的公共子序列
2. **确定递推公式**：当text2[i-1]==text1[j-1]时，dp[i][j]=dp[i-1][j-1]+1。否则dp[i][j]=max(dp[i-1][j],dp[i][j-1])
3. **dp数组初始化**:我们预留了第0列和第0行，最开始全部初始化为0即可
4. **确定遍历顺序** :从i=1，j=1开始向后遍历即可
5. **举例推导dp数组** ：如上



## 代码
```
class Solution {
public:
    int longestCommonSubsequence(string text1, string text2) {
        int m = text1.size();
        int n = text2.size();
        // 对于text2的[0,i)区间与text1的[0,j)区间的公共子序列
        vector<vector<int>> dp(n + 1, vector<int>(m + 1, 0));

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                if (text2[i - 1] == text1[j - 1]) // 末尾字母匹配
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                else // 末尾字母不匹配
                    dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
        return dp[n][m];
    }
};
```

时间复杂度O($n^2$)
空间复杂度O($n^2$)
