# 除了自身以外的数组乘积
[除了自身以外的数组乘积](https://leetcode.cn/problems/product-of-array-except-self/?envType=study-plan-v2&envId=top-100-liked)

## 前缀和
```
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        //可以使用前/后缀和（积）的方式来实现
        vector <int> pre(nums.size());
        vector <int> post(nums.size());
        vector <int> result(nums.size());
        pre[0]=nums[0];
        for(int i=1;i<nums.size();i++){
            pre[i]=nums[i]*pre[i-1];
        }
        post[nums.size()-1]=nums[nums.size()-1];
        for(int i=nums.size()-2;i>=0;i--){
            post[i]=nums[i]*post[i+1];
        }
        result[0]=post[1];
        result[nums.size()-1]=pre[nums.size()-2];
        for(int i=1;i<nums.size()-1;i++){
            result[i]=pre[i-1]*post[i+1];
        }
        return result;
    }
};
```
