# 买卖股票的最佳时机
[买卖股票的最佳时机](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/?envType=study-plan-v2&envId=top-100-liked)

正式进入了贪心算法的章节，贪心算法的核心思路就是利用局部最优解求解全局最优解，正如其名，够贪才能赢
贪心算法根据我的理解是那种根据直感可以实现但难以被理论证明的问题，这并不意味着贪心算法是简单的算法，相反困难的贪心题往往异常痛苦

我们本次来做稍微轻松一点的
## 解析
我们一直说低价买入高价卖出，在卖出价格相同时那肯定是用最少的钱买入最赚，所以其实很简单，我们提前记录下当前买入的最低价格就行

理解清楚思路相当简单的题
## 代码
```
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int result=0;

        int minprice=prices[0];//记录最便宜的时刻
        for(int i=1;i<prices.size();i++)
        {
            result=max(result,prices[i]-minprice);
            minprice=min(minprice,prices[i]);
        }

        return result;
    }
};
```

时间复杂度O(n)
空间复杂度O(1)
