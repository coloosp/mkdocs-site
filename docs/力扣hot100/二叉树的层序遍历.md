# 二叉树的层序遍历
[二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/?envType=study-plan-v2&envId=top-100-liked)

层序遍历我认为比起需要递归的其他遍历方式，思路其实好想很多，思路其实很简单，利用队列来作为辅助，访问并弹出队首元素，并在队尾加入其左右孩子（如果有的话）
我们添加了size与tmpsize作为判断当前层有多少节点的依据，这是作者喜欢的写法

话说层序遍历与广度优先搜索都用到了队列，并且它们的思想也相当接近，各位在学习图中的广度优先搜索时一定会有新的感悟的


```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> result;
        queue<TreeNode*> Q;
        if (root==nullptr)
            return result;

        int size = 1; // 当前层宽度
        int tmpsize;  // 下一层宽度
        TreeNode* t;  // 暂存节点
        Q.push(root);

        while (!Q.empty()) {
            tmpsize = 0;
            vector<int> level; // 搜集每一层节点
            // 搜集单层节点
            while (size--) {
                t = Q.front();
                Q.pop();
                level.push_back(t->val);

                if (t->left) {
                    Q.push(t->left);
                    tmpsize++;
                }
                if (t->right) {
                    Q.push(t->right);
                    tmpsize++;
                }
            }
            result.push_back(level);//加入收集到的单层节点
            size = tmpsize; // 更新下一层节点数
        }
        return result;
    }
};
```

时间复杂度O(n)
空间复杂度O(w) w为最大宽度
