# 回文链表
[回文链表](https://leetcode.cn/problems/palindrome-linked-list/?envType=study-plan-v2&envId=top-100-liked)


可以使用经典的快慢指针解决
总共分为三步
1. 利用快慢指针确定中间节点
2. 反转中间节点后面的链表
3. 从起点和中间判断后面的节点值是否相同

我们还会用到快慢指针很多次，请各位务必注意这种方法

```
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    bool isPalindrome(ListNode* head) {
         //使用快慢指针确定中间节点，再将中间后半部分反转，然后检查两部分即可
         //确定中间节点
         ListNode*slow=head;

         ListNode*fast=head;

         while(fast->next!=nullptr && fast->next->next!=nullptr){
            slow=slow->next;
            fast=fast->next;
            if(fast->next!=nullptr)
                fast=fast->next;
         }

         //slow现在会位于中间位置，将后面的链表反转
        //反转链表
         ListNode *p=slow->next;
         ListNode *t;
         slow->next=nullptr;
         while(p!=nullptr){
            t=p;
            p=p->next;
            t->next=slow->next;
            slow->next=t;
         }
        //确认回文链表
         ListNode *p1=head;
         ListNode*p2=slow->next;

         while(p2!=nullptr){
            if(p1->val !=p2->val)
                return false;
            p1=p1->next;
            p2=p2->next;
         }

         return true;
    }
};
```

时间复杂度O(n)
空间复杂度O(1)
