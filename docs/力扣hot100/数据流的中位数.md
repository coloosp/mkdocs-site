# 数据流的中位数
[数据流的中位数](https://leetcode.cn/problems/find-median-from-data-stream/description/?envType=study-plan-v2&envId=top-100-liked)

中位数和我们还真是有缘呢

这道题如果使用数组去实现，每次插入的时间复杂度是O(n)

各位也不要想着使用二分去偷鸡，因为即使我们确定了插入位置时间复杂度为O(logn)
我们插入元素一定是O(n),会被测试案例击毙的

## 解析
我想问大家一个问题，对于频繁插入的序列，该使用什么存储结构呢？我想出来的可能是树和堆

对于本题平衡二叉树应该也能够完成，但是平衡二叉树的插入和删除真不是普通人能够实现的

所以最终我们使用了堆，但是堆有一个问题，堆只能查找对于限定范围内前k个值。但是我们需要的n/2位置的值，我们根本就不能使用一个类似于解前面题的堆来完成

提示是比如这样的数组[1,2,3,4,5,6]
我们可以掰开成两半[1,2,3],[4,5,6]，我们只需要检查左边的尾部和右边的首部就可以获取中位数的值了

如果向这个数组加入数字7，我们约定先加入到左边，即将右边的4移动到
[1,2,3,4],[5,6,7] 我们只需要获取左边的尾部4即可

话都说到这里了，我们要使用的正是双堆结构，使用大顶堆左堆和小顶堆右堆来实现

因为在大顶堆中我们可以方便知道左边的尾部，在小顶堆中我们可以方便知道右边的首部

比如在这个例子[1,3,5] [6,7,9] 我们准备加入数组8到左堆
此时应该先加入8到右堆，调整过后[6,7,8,9]将6弹出并加入到左堆[1,3,5,6]

还有需要注意的是，虽然我们展示了清晰的数字，但是在堆中我们只可能知道堆顶的值

## 代码

```
class MedianFinder {
public:
    priority_queue<int> left;//左边大顶堆
    priority_queue<int,vector<int>,greater<>> right;//右边大顶堆


    MedianFinder() {
        //然而不需要实现什么
    }
    
    void addNum(int num) {
        //左右相等时优先插入左堆
        if(left.size()==right.size())
        {
            //先将值插入右堆整理出堆顶值后插入左堆
            right.push(num);
            left.push(right.top());
            right.pop();
        }
        else
        {
            //同样的操作
            left.push(num);
            right.push(left.top());
            left.pop();
        }
    }
    
    double findMedian() {
        //偶数时取两个堆顶的平均值
        if(left.size()==right.size())
        {
            return (left.top()+right.top())/2.0;
        }
        else
            return left.top();
    }
};
```
插入时间复杂度O(logn)
空间复杂度O(n)

```
class MedianFinder {
public:

    vector<int> nums;
    MedianFinder() {
        //然而并不需要做什么
    }
    
    void addNum(int num) {
        if(nums.size()==0)
            {
                nums.push_back(num);
                return;
            }
        int low=0;
        int high=nums.size()-1;

        while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid] > num)
            {
                high=mid-1;
            }
            else
                low=mid+1;
        }

        nums.resize(nums.size()+1);
        for(int i=nums.size()-1;i>low;i--)
        {
            nums[i]=nums[i-1];
        }        
        nums[low]=num;
    }
    
    double findMedian() {
        if(nums.size() %2!=0)
        {
            return nums[nums.size()/2];
        }
        else
        {
            return ( nums[nums.size()/2-1] +nums[nums.size()/2] )/2.0;
        }
    }
};

```

插入时间复杂度O(n)
空间复杂度O(n)
