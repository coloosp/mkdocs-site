# 二叉树的右视图
[二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/?envType=study-plan-v2&envId=top-100-liked)

这道题也是属于有点小巧思的题目，看起来很复杂，但我们实际上有方法解决的

也就是使用层序遍历，获取每一层的最后一个节点
当然力扣评论区的大家也想到了这种简单方法

## 层序遍历
作者甚至直接套用了二叉树的层序遍历的代码
```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        //按道理使用层序遍历然后获取每一层最后一个节点即可
        //这应该是我想出的最简单的方式了
        vector<int> result;
         vector<vector<int>> levels;
        queue<TreeNode*> Q;
        if (root==nullptr)
            return result;

        int size = 1; // 当前层宽度
        int tmpsize;  // 下一层宽度
        TreeNode* t;  // 暂存节点
        Q.push(root);

        while (!Q.empty()) {
            tmpsize = 0;
            vector<int> level; // 搜集每一层节点
            // 搜集单层节点
            while (size--) {
                t = Q.front();
                Q.pop();
                level.push_back(t->val);

                if (t->left) {
                    Q.push(t->left);
                    tmpsize++;
                }
                if (t->right) {
                    Q.push(t->right);
                    tmpsize++;
                }
            }
            levels.push_back(level);//加入收集到的单层节点
            size = tmpsize; // 更新下一层节点数
        }
        
        for(vector<int> level :levels)
        {
            result.push_back(level[level.size()-1]);
        }
        return result;

    }
};
```

时间复杂度O(n)
空间复杂度O(n)

## 深度优先搜索(前序遍历)
这是力扣大佬灵茶山艾府的思路，相当巧妙，各位如果有不会的题目也可以看看灵神的题解
前序遍历的本质就是深搜，这里理解为深搜反而比前序遍历更好

```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left),
 * right(right) {}
 * };
 */
class Solution {
public:
    vector<int> result;
    void dfs(TreeNode* node, int depth) {
        if (node == nullptr)
            return;
        if (depth == result.size()) { //当首次遇见该节点时
            result.push_back(node->val);
        }
        dfs(node->right, depth + 1); //先深搜右子树，可以保证右视图是右边优先的
        dfs(node->left, depth + 1);
    }
    vector<int> rightSideView(TreeNode* root) {
        dfs(root, 0);
        return result;
    }
};
```
时间复杂度O(n)
空间复杂度O(n)
虽然看起来两种时间空间复杂度都一样，但代码简洁性这块被完爆了
