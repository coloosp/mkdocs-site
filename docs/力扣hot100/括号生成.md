# 括号生成
[括号生成](https://leetcode.cn/problems/generate-parentheses/description/?envType=study-plan-v2&envId=top-100-liked)

这看起来是和前面的题不太一样，我一开始看也没有思路，不过，抓住问题要我们做什么，“加入括号”是我们需要做的，加入什么样的括号？只有两种，左括号和右括号

## 解析
这样就很好理解了，我们吧左括号理解为数字"1"右括号理解为数字"2" 我们每次抓一个数字进入s_path中，不过这是有条件的，右括号数量不能先大于左括号，也就是不能出现"))(("这种

其实很好理解了，就是可以无限使用元素的组合，这个搜索数组只有两个值["(",")"]

回溯递归三步：
1. 确定参数，输出要求匹配数n，当前左括号数量left,当期右括号数量right
2. 当存储了2*n个字符说明满足条件，加入结果集并返回
3. 你可以抽象理解为`for(int i=0;i<2;i++)`我们每一次抓取一个括号加入，并且right不能大于left

## 代码
```
class Solution {
public:
    vector <string> result;
    string s_path;
    //left、right分别表示左右括号存在的个数
    void backtracking(int n,int left,int right)
    {
        if(s_path.size()==2*n) //满足条件
        {
            result.push_back(s_path);
            return;
        }

        //左括号还没装满
        if(left < n)
        {
            s_path+='(';
            backtracking(n,left+1,right);
            s_path.pop_back();
        }

        //右括号小于左括号数量
        if(right <left && right<n)
        {
            s_path+=')';
            backtracking(n,left,right+1);
            s_path.pop_back();
        }
    }


    vector<string> generateParenthesis(int n) {
        backtracking(n,0,0);
        return result;
    }
};
```

```
class Solution {
public:
    vector <string> result;
    string s_path;
    //left、right分别表示左右括号存在的个数
    void backtracking(int n,int left,int right)
    {
        if(s_path.size()==2*n) //满足条件
        {
            result.push_back(s_path);
            return;
        }

       for(int i=0;i<2;i++)
       {
        if(i==0&&left<n)
        {
            s_path+='(';
            backtracking(n,left+1,right);
            s_path.pop_back();
        }

        if(i==1 && right <left)
        {
            s_path+=')';
            backtracking(n,left,right+1);
            s_path.pop_back();
        }
       }  
    }


    vector<string> generateParenthesis(int n) {
        backtracking(n,0,0);
        return result;
    }
};
```
