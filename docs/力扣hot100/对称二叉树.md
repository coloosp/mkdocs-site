# 对称二叉树
[对称二叉树](https://leetcode.cn/problems/symmetric-tree/?envType=study-plan-v2&envId=top-100-liked)

各位可能试着像前面的思路一样，递归比较节点的左子树和右子树，实际上并不能这样做，各位看看案例一就知道，这样的比较并非是对称二叉树

真正的比较应该是这样做
1. 传入左节点与右节点是否相同
2. 左节点的左孩子与右节点的右还在是否相同
3. 左节点的右孩子与右节点的左孩子是否相同

对于左右节点的比较也有说法
1. 左节点为空右节点不为空，返回false
2. 左节点不为空，右节点为空，返回false
3. 左右节点都为空,返回true
4. 左右节点值不相同，返回false
5. 左右节点值相同，继续下一步操作

```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    bool compare(TreeNode* left,TreeNode* right)
    {
        if(left==nullptr && right)
            return false;
        else if(left && right==nullptr)
            return false;
        else if(left==nullptr && right==nullptr)
            return true;
        else if(left->val!=right->val) //注意要把这个放在下面否则会空指针错误
            return false;
        bool lf=compare(left->left,right->right); //判断左子树的左子树与右子树的右子树
        bool rt=compare(left->right,right->left);//判断左子树的右子树和右子树的左子树
        return lf&& rt; //总结左右子树情况
    }
    bool isSymmetric(TreeNode* root) {
        return compare(root->left,root->right);
    }
};
```
