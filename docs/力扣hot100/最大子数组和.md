# 最大子数组和
[最大子数组和](https://leetcode.cn/problems/maximum-subarray/description/?envType=study-plan-v2&envId=top-100-liked)

## 前缀和
使用前缀和或者滑动窗口应该是求这种连续子串的常见思路，明显使用前缀和更加优秀
```
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        //可以使用前缀和来完成这道题
        //思路如下，我们统计前缀和，可以知道连续子数组的结果应该是一个前缀和减去另一个前缀和
        //所以知道i的最大和连续子数组和一个为当前前缀和减去之前的最小前缀和
        int presum=0;
        int minpresum=0;//这里确实需要初始化为0，要保证比如只有负数的数组和只有一个数字的数组
        int result=INT_MIN;
        for(int i=0;i<nums.size();i++){
            presum+=nums[i];
            result=max(result,presum-minpresum);
            minpresum=min(minpresum,presum);//需要把更新放在后面，因为我们需要计算的是之前的一个前缀和

        }
        return result;

    }
};
```

时间复杂度O(n)
空间复杂度O(1)

## 贪心
贪心的意思就是不要拖后腿，也就是不能接受前缀和是负数的情况
```
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        //可以使用贪心的思路来实现
        //可以设想，如果前面的数是正数，我们肯定乐意加上，如果是负数，那肯定是避之不及

        int result=INT_MIN;
        int ans=0;
        for(int i=0;i<nums.size();i++){
            ans+=nums[i];//尝试加入当前数字

            result=max(ans,result);//更新最值

            if(ans <0) ans=0;//重置的意思是我们应该从下一个下标开始计算了，因为前面已经出现负数
        }

        return result;

    }
};
```

时间复杂度O(n)
空间复杂度O(1)
