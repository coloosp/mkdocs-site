# 岛屿数量
[岛屿数量](https://leetcode.cn/problems/number-of-islands/description/?envType=study-plan-v2&envId=top-100-liked)

欢迎来到图论的学习，关于图论力扣100居然只有四道题(不过都是经典算法)，实际上图论还是相当难的章节，即使各位学习过数据结构的图论也不一定能够很轻松完成

说到图论应该逃不过广度优先搜索和深度优先搜索吧，本题属于图论超级经典题，让我们使用熟悉的广搜和深搜来一一实现吧!

## 广度优先搜索
### 解析
对于广搜一般是这样的逻辑
1. 构建存储坐标节点的队列，标记是否已经搜索过的标记数组
2. 确定符合条件的节点并进入广搜，加入且标记该节点然后进入广搜循环
3. 当队列非空时弹出队首，加入与其相邻且符合条件的其他节点入栈并标记，直到栈空为止

不过对于本题有很多细节不小心的话很容易出错
1. 下面代码中x是行坐标对应边界m，y为列坐标则对应n
2. 需要先判断是否超界再判断节点是否符合条件
3. 需要在节点入队时标记而不是弹出队列时标记，因为这样会错误地判断很多重复节点影响效率
4. 对于广搜函数传入的所有数组请引用传入，一是因为引用节省内存，二是因为比如标记数组需要被修改

### 代码
```
class Solution {
public:
    int direct[4][2]={{1,0},{-1,0},{0,1},{0,-1}}; 

    void BFS(vector<vector<char>>&grid,vector<vector<int>>&used, int x,int y,int m,int n)
    {
        queue <pair<int,int>> Q; //辅助队列
        Q.push({x,y});
        used[x][y]=1;
        //记住是入队时标记
        while(!Q.empty())
        {
            pair<int,int> node;
            node=Q.front();
            Q.pop();
            int curx=node.first;
            int cury=node.second;
            for(int i=0;i<4;i++)
            {
                int newx=curx+direct[i][0];
                int newy=cury+direct[i][1];

                //注意要提前判断超界条件
                //注意x表示行坐标 y表示列坐标
                if(newx >=m || newx<0 || newy >=n || newy <0)
                    continue;
                
                //只加入未标记的陆地
                if(used[newx][newy]==0 && grid[newx][newy]=='1'){
                used[newx][newy]=1;
                Q.push({newx,newy});
                }
            }
        }
    }

    int numIslands(vector<vector<char>>& grid) {
        //使用广度优先搜索实现吧！
        //广度优先搜索是利用队列加入一个陆地，我们可以标记这个陆地是已经使用过的
        int m=grid.size();
        int n=grid[0].size();
        vector<vector<int>> used(m,vector<int>(n,0)); //标记数组
        int result=0;

        for(int i=0;i<m;i++)
        {
            for(int j=0;j<n;j++)
            {
                //检查到未标记的陆地
                if(grid[i][j]=='1' && used[i][j]==0)
                {
                    result++;
                    BFS(grid,used,i,j,m,n);
                }
            }
        }
        return result;
    }
};
```
时间复杂度O(m*n)

空间复杂度O(m*n)
## 深度优先搜索
各位可以想象(或者在网上看看视频)，就知道深搜和广搜的区别与运作条件了，虽然广搜可能会更容易理解，但是深搜在本题代码量明显少了，深搜完成的步骤是这样的
1. 构建标记数组
2. 选择符合条件的节点进入深搜，标记该节点并且选择一个与其相邻的符合条件的节点进入下一次深搜
3. 重复以上深搜直到无法在继续，void类型的深搜会自动结束

```
class Solution {
public:
    int direct[4][2]={{1,0},{-1,0},{0,1},{0,-1}}; //方向数组
    void DFS(vector<vector<char>>& grid,vector<vector<int>>&used,int x,int y,int m,int n)
    {
        used[x][y]=1;
        for(int i=0;i<4;i++)
        {
            int newx=x+direct[i][0];
            int newy=y+direct[i][1];

            if(newx <0 || newx >=m ||newy <0 || newy >=n)
                continue;
            if(grid[newx][newy]=='1' && used[newx][newy]==0)
                DFS(grid,used,newx,newy,m,n);
        }
    }
  

    int numIslands(vector<vector<char>>& grid) {
        //使用深度优先搜索来实现吧！
        int m=grid.size();
        int n=grid[0].size();
        vector<vector<int>> used(m,vector<int>(n,0)); //标记数组
        int result=0;

        for(int i=0;i<m;i++)
        {
            for(int j=0;j<n;j++)
            {
                //检查到未标记的陆地
                if(grid[i][j]=='1' && used[i][j]==0)
                {
                    result++;
                    DFS(grid,used,i,j,m,n);
                }
            }
        }
        return result;
    }
};
```
时间复杂度O(m*n)

空间复杂度O(m*n)

深搜和广搜各有优劣，请根据实际情况来选择合适的搜索方式
