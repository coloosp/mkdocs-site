# 最小路径和

[最小路径和](https://leetcode.cn/problems/minimum-path-sum/?envType=study-plan-v2&envId=top-100-liked)

和前面相当一致的思路，各位理解了前面也能很好完成这道题
## 解析
注意这里我们并没有预留第0行和第0列，这是为了匹配grid的下标，总之各位觉得怎么简单怎么写就行了
1. **确定dp数组与下标的含义**:dp[i][j]表示到(i,j)的最小路径和
2. **确定递推公式**: dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];选择上方或者左方的更小值
3. **dp数组初始化**:为了避免超界问题提前更新好第0行和第0列
4. **确定遍历顺序**：从左上角更新到右下角就行 
5. **举例推导dp数组**

|  1  |  4  |  5  |
| --- | --- | --- |
| 2   | 7   | 6   |
| 6   | 8   | 7   |

多得就不说了，和前面思路几乎一致

## 代码
```
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        // 按道理也可以使用深搜，我们就不重要使用了
        int m = grid.size();
        int n = grid[0].size();
        vector<vector<int>> dp(m, vector<int>(n, INT_MAX));
        dp[0][0] = grid[0][0];

        for (int i = 1; i < n; i++)
            dp[0][i] = dp[0][i - 1] + grid[0][i];

        for (int i = 1; i < m; i++)
            dp[i][0] = dp[i - 1][0] + grid[i][0];

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++)
                dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];//更新为左边或者上边的更小值+当前值
        }

        return dp[m - 1][n - 1];
    }
};
```

时间复杂度O($n^2$)
空间复杂度O($n^2$)
