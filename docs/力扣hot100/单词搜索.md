# 单词搜索
[单词搜索](https://leetcode.cn/problems/word-search/?envType=study-plan-v2&envId=top-100-liked)

实际上是深搜+回溯的模板题，相当值得学习！

## 解析
各位可以看看[岛屿数量](https://leetcode.cn/problems/number-of-islands/?envType=study-plan-v2&envId=top-100-liked)
回忆一下深搜是怎么实现的
想想我们在深搜时都在做什么，emmm
标记坐标，更新坐标，进入下一层坐标
这样有一个问题是，我们就只能一条路走到死，当我们的一条深搜标记了used之后，其他深搜路径就不能搜索了
我们又得使用used数组防止兜圈子，又得避免一次深搜标记后下一次不能搜索，顺理成章地需要使用回溯的思想，也就是在函数即将退栈时恢复搜索路径s_path与used

另外我们使用了剪枝以及判断结果的位置也需要注意

我使用了flag来标记是否存在正确路径，这样可以更好理解与实现

## 代码

```
class Solution {
public:
    int dir[4][2] = {{1, 0}, {-1, 0}, {0, 1}, {0, -1}}; // 居然是方向数组
    bool flag = false; //用于标记是否匹配成功
    string s_path;
    void dfs(vector<vector<char>>& board, vector<vector<int>>& used,
             string word,int x,int y,int m,int n) {
        
        if(flag)
            return;

        if(board[x][y]!=word[s_path.size()]) //剪枝，说明下一个加入的已经不符合了
            return;


        s_path+=board[x][y];//加入当前字符
        used[x][y]=1;

        //这里确实要放在后面，前面的判断已经判断过加入的字符是否符合条件了
        if(s_path.size()==word.size())
            {
                flag=true;
                return;
            }

        for(int i=0;i<4;i++)
        {
            int newx=x+dir[i][0];
            int newy=y+dir[i][1];

            if(newx >=m ||newx <0 || newy >=n || newy <0) //别忘了先判断超界
                continue;
            
            if(used[newx][newy]==0)
            {
                dfs(board,used,word,newx,newy,m,n);
            }

        }

        s_path.pop_back();
        used[x][y]=0;//恢复现场   

    }

    bool exist(vector<vector<char>>& board, string word) {
        // 你觉得你是深搜吗？
        int m = board.size();
        int n = board[0].size();
        vector<vector<int>> used(m, vector<int>(n, 0));
        for(int i=0;i<m;i++)
        {
            for(int j=0;j<n;j++)
                dfs(board,used,word,i,j,m,n);
        }

        return flag;
    }
};
```
