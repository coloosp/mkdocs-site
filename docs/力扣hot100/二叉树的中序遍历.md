# 二叉树的中序遍历
[二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/description/?envType=study-plan-v2&envId=top-100-liked)

终于进入二叉树的章节了，二叉树也是数据结构相当重要的一项

嘛，对于这种题应该说是知道的人不用讲，不知道的人讲不清楚，各位只要学习过二叉树应该对三种遍历的递归形式很熟悉了吧

## 递归法
没什么好讲的，各位一定也清楚这种方式
```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    void inorder(TreeNode* node,vector<int> &result)
    {
        if(node==nullptr)//空节点返回
            return;

        inorder(node->left,result);//搜索左子树
        result.push_back(node->val);//读取中间节点
        inorder(node->right,result);//搜索右子树
    }


    vector<int> inorderTraversal(TreeNode* root) {
        //中序遍历就是左中右
        vector <int> result;
        inorder(root,result);
        return result;
    }
};
```
时间复杂度O(n)
空间复杂度O(logn) 函数栈开销

## 迭代法
各位的迭代法可能都有所不同，这里说说我写的迭代法的原理
根据中序遍历的思路，我们应该尽可能访问左的方向，按照左中右的思路，到最左端会出现两种情况
1. 只有一个左叶子节点
2. 最左端，但是有右子树

这就很好处理了，我们可以利用栈的思维，先从根节点向左搜索并加入到栈中，此时我们可以通过栈知道每一层的左端并且可以通过这个左端知道其右子树信息
当我们加入到无法加入时，弹出栈顶，这是最左端的节点，并且将工作指针更新为其右子树(最左端的右子树比上一层更"左"没错吧)

其中细节处理可能需要各位再思考，如果无法理解也没有关系，这属于经典算法，多练习几遍肯定能掌握的
```
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> result;
        stack<TreeNode*> st;
        TreeNode* cur=root;//工作指针
        TreeNode* t;//暂存指针
        while(cur!=nullptr || !st.empty())
        {
            if(cur) //如果工作指针存在
            {
                st.push(cur);
                cur=cur->left;
            }
            else //说明向左搜索结束，向上向右返回
            {
                t=st.top();
                st.pop();
                result.push_back(t->val); //获取中间节点
                cur=t->right;//搜索右子树
            }
        }
         return result;
    }
};
```

时间复杂度O(n)
空间复杂度O(logn) 显示栈开销
