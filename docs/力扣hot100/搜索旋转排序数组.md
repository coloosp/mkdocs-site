# 搜索旋转排序数组
[搜索旋转排序数组](https://leetcode.cn/problems/search-in-rotated-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

这道题根据直感来说，一般会采取以下的思路
* 找到两段序列的边界
* 根据边界查找对应序列的目标值下标

## 解析
因此找到边界就是我们最关键的任务，也就是找到最小值
![](vx_images/482911123566477.png)

要小心这种情况，即下标在0开始进行旋转数组，这时相当于原数组，我们可以认为也是第二段的数组
根据last值2进行的二分查找可以有以下判断
* nums[mid]>last，说明还在第一段，low=mid+1
* nums[mid]<=last,说明在第二段，high=mid-1

最终的low就是最小值下标index了

那target就很好找了，我们也可以根据last判断target在第一段还是第二段

## 代码

```
class Solution {
public:
    int findmid(vector<int>&nums)
    {
        int last=nums[nums.size()-1];
        int low=0;
        int high=nums.size()-1;
        while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]>last)//在第一段
                low=mid+1;
            else //在第二段或者本身就是递增序列
                high=mid-1;
        }
        return low;//low为最小值下标
    }

    //经典二分查找
    int binarysearch(vector<int>&nums,int left,int right,int target)
    {
        int low=left;
        int high=right;
        while(low<=high)
        {
            int mid=(low+high)/2;

            if(nums[mid]==target)
                return mid;

            if(nums[mid]>target)
                high=mid-1;
            else
                low=mid+1;
        }
        return -1;
    }

    int search(vector<int>& nums, int target) {
        //本题使用两步完成，找到两数组边界(即最小值)，根据最小值分割区间并在对应序列中找到目标值

        int index=findmid(nums);//最小值下标
        int last=nums[nums.size()-1];//右端边界值

        if(target>last) //在第一段
            return binarysearch(nums,0,index-1,target);
        else
            return binarysearch(nums,index,nums.size()-1,target);
    }
};
```

时间复杂度O(logn)
空间复杂度O(1)
