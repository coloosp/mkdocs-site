## 解析
除了动态规划我们还可以使用什么？答案是栈，我们在之前就实现栈的括号匹配，如果使用栈的话这道题甚至只是中等题

对于对于栈的括号匹配我们一般会做这样的事情，如果遇到'('我们加入栈中，如果遇到')'我们弹出栈顶的左括号,注意我们只能弹出栈顶的左括号，否则就并非连续的子串了

比如像"()))"结果应该是2，所以思想应该是一个左括号对应一个右括号，如果栈顶是左括号那一切好说，但如果栈顶是右括号怎么办？答案是我们需要重新开始计算，这时候已经"断连"了

我们向栈中存储的是下标(是不是很像单调栈的写法？)，如果遇到')'我们弹出栈顶左括号，接下来栈顶是我们连续子串的前驱(或者说叫基准)，也就是说当前我们连续子串应该是这样的[st.top()+1，i]，为了方便比如()的计算我们还应该加入-1作为基准
我们知道如果是左括号我们正常弹出，没什么好说的，比如(() 我们知道结果应该是2
但())()这种怎么办？
我们来看看该怎么更新
`-1`
`-1,0`
`-1`弹出0，计算结果为1-(-1)=2
`2`这时候已经断连，我们需要更新基准
`2,3`
`2`弹出3，计算结果为4-2=2

再复现一遍
* 基准：遍历到连续子串端的前驱
* 遇到`(`加入下标
* 遇到`)`如果此时栈顶为左括号，栈应该大小大于1，弹出栈顶根据接下来栈顶基准计算当前连续子串长度
* 遇到`)`如果此时栈顶为右括号，此时已经断连，需要更新基准，弹出旧基准，加入当前下标作为新基准

我尽量把我的理解写了出来，但可能还讲的不太明白，各位也可以去看看其他有图例的题解，作者精力有限抱歉

## 代码
```
class Solution {
public:
    int longestValidParentheses(string s) {
        //看到括号匹配就会想到栈
        stack <int> st;
        st.push(-1);//加入初始基准

        int result=0;
        for(int i=0;i<s.size();i++)
        {
            if(s[i]=='(')
                st.push(i);
            else if(s[i]==')' && st.size()>1)//栈顶是左括号正常处理
            {
                st.pop();
                result=max(result,i-st.top());
            }
            else//说明栈顶是右括号，更新基准
            {
                st.pop();
                st.push(i);
            }
        }
        return result;

    }
};
```

时间复杂度O(n)
空间复杂度O(n)
