## 解析


如何分割字符串？实际上是在问：在哪些地方切割，切几段？
怎么做？暴力试!回溯就是干这个的，回溯的本质就是利用函数递归来实现暴力算法

回文判断相信各位在大学课堂已经见过很多次了
这里说说C++的坑b stl函数substr
s.substr(int start,int len)
第一个参数是子串起点下标，第二个参数是子串长度，我们要取[index,i]的字符串时应该是i-index+1

再说说递归回溯三步吧
1. 确定参数：字符串s与当前起点index
2. 确定终止条件：当index==s.size()时后续没有任何需要切割的子串了
3. 单层递归逻辑:确定[index,i]是否为回文，是回文时切割并加入path，下一次从i+1开始切割

## 代码
```
class Solution {
public:
    vector<vector<string>> result;
    vector<string> path;

    //判断回文
    bool isPalindrome(string&s,int low,int high)
    {
        for(int i=low,j=high;i<j;i++,j--)
        {
            if(s[i]!=s[j])
                return false;
        }
        return true;
    }
    void backtracking(string&s,int index)
    {
        if(index==s.size())//到这里说明已经没有需要切割的子串了
        {
            result.push_back(path);
            return;
        }

        for(int i=index;i<s.size();i++)
        {
            if(isPalindrome(s,index,i))
            {
                string str=s.substr(index,i-index+1);//切割子串1
                path.push_back(str);
                backtracking(s,i+1);
                path.pop_back();
            }
        }
    }


    vector<vector<string>> partition(string s) {
        //套路一样的小把戏罢了
        backtracking(s,0);
        return result;
    }
};
```
